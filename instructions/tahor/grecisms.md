## 🔥 ВВЕДЕНИЕ

Грецизмы — это слова, которые пришли из греческого языка через Септуагинту и апостольские писания. Греческая философия заменила ивритскую конкретику. Абстракция заменила образ. Идея заменила действие.

Здесь собраны все известные грецизмы с заменами.

## 📜 СПИСОК

- **Агапэ** (ἀγάπη) → אַהֲבָה (аhава) — любовь, преданность
- **Ангел** (ἄγγελος) → מַלְאָךְ (малъах) — вестник, посланник
- **Апокалипсис** (ἀποκάλυψις) → גִּלּוּי (гилуй) — раскрытие, откровение
- **Апостол** (ἀπόστολος) → שָׁלִיחַ (шалиах) — посланник, уполномоченный
- **Архиерей** (ἀρχιερεύς) → כֹּהֵן גָּדוֹל (коhэн гадоль) — великий служитель
- **Бес** (δαίμων) → שֵׁד (шед) — разрушитель; או רוּחַ טָמֵא (руах тамэ) — дыхание непригодное
- **Библия** (βιβλία) → כִּתְבֵי הַקֹּדֶשׁ (китвей hа-кодеш) — Писания отделённости; или ТаНаХ
- **Гигант** (γίγας) → גִּבּוֹר (гибор) — сильный, могучий; или נָפִיל (нафиль) — падший
- **Герой** (ἥρως) → גִּבּוֹר (гибор) — сильный, могучий
- **Диавол** (διάβολος) → שָׂטָן (сатан) — противник; או מַלְשִׁין (мальшин) — клеветник
- **Диакон** (διάκονος) → מְשָׁרֵת (мешарэт) — служитель, прислужник
- **Догмат** (δόγμα) → חֹק (хок) — устав; או מִצְוָה (мицва) — связь
- **Дракон** (δράκων) → תַּנִּין (танин) — морское чудовище, змей
- **Евангелие** (εὐαγγέλιον) → בְּשׂוֹרָה (бсора) — благая весть
- **Евхаристия** (εὐχαριστία) → תּוֹדָה (тода) — благодарение; או קָרְבַּן תּוֹדָה (корбан тода) — жертва благодарения
- **Епископ** (ἐπίσκοπος) → פָּקִיד (пакид) — надзиратель, блюститель
- **Епитимья** (ἐπιτιμία) → עֹנֶשׁ (онеш) — последствие; нет прямого аналога
- **Ересь** (αἵρεσις) → כַּת (кат) — группа, течение
- **Еретик** (αἱρετικός) → מִין (мин) — отступник; או פּוֹרֵשׁ (пореш) — отделившийся
- **Жертвенник** (θυσιαστήριον) → מִזְבֵּחַ (мизбеах) — место жертвоприношения
- **Идол** (εἴδωλον) → פֶּסֶל (пэсэль) — истукан, изваяние; או אֱלִיל (элиль) — ничтожный
- **Идолопоклонство** (εἰδωλολατρεία) → עֲבוֹדָה זָרָה (авода зара) — служение чужому
- **Иерей** (ἱερεύς) → כֹּהֵן (коhэн) — служитель
- **Икона** (εἰκών) → צֶלֶם (целем) — образ; запрещено для поклонения
- **Исповедник** (ὁμολογητής) → מוֹדֶה (модэ) — признающий; нет аналога
- **Канон** (κανών) → מִשְׁפָּט (мишпат) — порядок; או חֹק (хок) — устав
- **Кафолический** (καθολικός) → כְּלָלִי (клали) — всеобщий; не относится к собранию
- **Келейно** (κελλίον) → בְּיִחִידוּת (бе-йехидут) — в уединении; нет аналога
- **Клирик** (κληρικός) → מְשָׁרֵת (мешарэт) — служитель
- **Космос** (κόσμος) → עוֹלָם (олам) — скрытое, далёкое время; או תֵּבֵל (тэвэль) — обитаемый мир
- **Крест** (σταυρός) → עֵץ (эц) — дерево; או תְּלִיָּה (тлия) — повешение
- **Крещение** (βάπτισμα) → טְבִילָה (твила) — погружение, омовение
- **Лавра** (λαύρα) → מוֹשַׁב נָזִירִים (мошаб назирим) — поселение отделённых; нет аналога
- **Лик** (λαός) → מַחֲנֶה (махане) — стан; או קָהָל (каhал) — община
- **Литургия** (λειτουργία) → עֲבוֹדָה (авода) — служение
- **Логос** (λόγος) → דָּבָר (давар) — слово, дело
- **Миро** (μύρον) → שֶׁמֶן הַמִּשְׁחָה (шемен hа-мишха) — масло помазания
- **Миссия** → שְׁלִיחוּת (шлихут) — посланничество
- **Монастырь** (μοναστήριον) → מוֹשַׁב נְזִירִים (мошаб назирим) — поселение отделённых
- **Монах** (μοναχός) → נָזִיר (назир) — отделённый, посвящённый
- **Мученик** (μάρτυς) → нет прямого аналога
- **Омофор** (ὠμοφόριον) → כְּסוּת (кесут) — покрытие; нет аналога
- **Паремия** (παροιμία) → מָשָׁל (машал) — притча, иносказание
- **Патриарх** (πατριάρχης) → אָב (ав) — отец; или רֹאשׁ אָבוֹת (рош авот) — глава отцов
- **Пневма** (πνεῦμα) → רוּחַ (руах) — дыхание, ветер
- **Помазание** (χρίσμα) → מִשְׁחָה (мишха) — помазание
- **Пост** (νηστεία) → צוֹם (цом) — пост
- **Пресвитер** (πρεσβύτερος) → זָקֵן (закен) — старейшина
- **Притча** (παραβολή) → מָשָׁל (машал) — иносказание, притча
- **Пророк** (προφήτης) → נָבִיא (нави) — вестник, призванный
- **Пророчество** (προφητεία) → נְבוּאָה (невуа) — вестничество
- **Просфора** (προσφορά) → לֶחֶם הַקָּרְבָּן (лехем hа-корбан) — хлеб приближения
- **Псалом** (ψαλμός) → מִזְמוֹר (мизмор) — песнь
- **Псюхэ** (ψυχή) → נֶפֶשׁ (нэфеш) — дышащее существо
- **Равноапостольный** (ἰσαπόστολος) → כִּשְׁלִיחַ (ки-шлиах) — как посланник; нет аналога
- **Святитель** (ἱεράρχης) → מוֹרֶה (морэ) — учитель; нет аналога
- **Символ** (σύμβολον) → אוֹת (от) — знак
- **Скит** (σκήτη) → מִדְבָּר (мидбар) — пустыня; או מָקוֹם נָזִיר (маком назир) — место отделённого
- **Соборность** (καθολικότης) → אַחְדוּת (ахдут) — единство; нет аналога
- **Стихира** (στιχηρόν) → פִּיּוּט (пиют) — гимн; нет аналога
- **Схизма** (σχίσμα) → פִּלּוּג (пилуг) — разделение, раскол
- **Схима** (σχῆμα) → צוּרָה (цура) — образ; нет аналога
- **Титан** (Τιτάν) → גִּבּוֹר (гибор) — сильный; или נָפִיל (нафиль) — падший
- **Тропарь** (τροπάριον) → פִּיּוּט (пиют) — гимн; нет аналога
- **Фанатик** (φανάτικός) → קַנַּאי (канай) — ревнитель
- **Фимиам** (θυμίαμα) → קְטֹרֶת (кеторет) — курение
- **Философия** (φιλοσοφία) → חָכְמָה (хохма) — мудрость
- **Хаос** (χάος) → תֹּהוּ וָבֹהוּ (тоhу ва-воhу) — пустота и безвидность
- **Хартия** (χάρτης) → מְגִלָּה (мегила) — свиток
- **Херувимская** (χερουβικόν) → שִׁיר הַכְּרוּבִים (шир hа-керувим) — песнь керувимов
- **Хиротония** (χειροτονία) → סְמִיכָה (смиха) — возложение рук
- **Хорепископ** (χωρεπίσκοπος) → פָּקִיד הַכְּפָר (пакид hа-кфар) — надзиратель села
- **Христос** (Χριστός) → מָשִׁיחַ (Машиах) — Помазанник
- **Экзарх** (ἔξαρχος) → נָשִׂיא (наси) — глава; нет аналога
- **Экклесия** (ἐκκλησία) → קְהִלָּה (кеhила) — собрание, община
- **Эон** (αἰών) → עוֹלָם (олам) — скрытое, далёкое время
- **Эпитрахиль** (ἐπιτραχήλιον) → אַבְנֵט (авнет) — пояс; нет аналога
- **Эсхатология** (ἐσχατολογία) → אַחֲרִית הַיָּמִים (ахарит hа-ямим) — конечное время
