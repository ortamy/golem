## 🔥 ВВЕДЕНИЕ

Латинизмы — это слова, которые пришли из латыни через Вульгату и католическую традицию. Рим не переводил — он импортировал. Латинские слова становились «священными терминами», которые никто не смел трогать.

Здесь собраны все известные латинизмы с заменами.

## 📜 СПИСОК

- **Алтарь** (altare) → מִזְבֵּחַ (мизбеах) — жертвенник
- **Вероучение** (doctrina) → יְסוֹדוֹת הָאֱמוּנָה (йесодот hа-эмуна) — основы верности; или תּוֹרָה (тора) — наставление
- **Гимн** (hymnus) → שִׁיר (шир) — песнь; или תְּהִלָּה (тэhила) — хвала
- **Грех** (peccatum) → חֵטְא (хэт) — промах
- **Диакон** (diaconus) → מְשָׁרֵת (мешарэт) — служитель
- **Догмат** (dogma) → אֱמוּנָה (эмуна) — верность (как действие); או תּוֹרָה (тора) — наставление
- **Епископ** (episcopus) → פָּקִיד (пакид) — надзиратель; או מְבַקֵּר (мевакер) — смотрящий
- **Ересь** (heresis) → כַּת (кат) — группа, течение
- **Жертва** (sacrificium) → קָרְבָּן (корбан) — приближение
- **Индульгенция** (indulgentia) → нет аналога в ТаНаХе. Практика отпуска промахов за плату чужда Торе
- **Исповедание** (professio) → הוֹדָיָה (ходая) — признание, благодарение
- **Исповедь** (confessio) → וִדּוּי (видуй) — признание
- **Кардинал** (cardinalis) → нет аналога в ТаНаХе. Иерархический титул римского собрания
- **Клир** (clerus) → אַנְשֵׁי הַקְּהִלָּה (аншей hа-кеhила) — люди собрания; нет прямого аналога
- **Крест** (crux) → עֵץ (эц) — дерево; או אוֹת (от) — знак
- **Крестный ход** (processio) → הַלְכָה (hалха) — шествие; нет прямого аналога
- **Крещение** (baptisma) → טְבִילָה (твила) — погружение
- **Ладан** (thus) → קְטֹרֶת (кеторет) — курение
- **Литургия** (liturgia) → עֲבוֹדָה (авода) — служение
- **Месса** (missa) → нет аналога в ТаНаХе. Жертвоприношение, заменяющее жертву Машиаха
- **Миссионер** (missionarius) → שָׁלִיחַ (шалиах) — посланник
- **Монах** (monachus) → נָזִיר (назир) — отделённый
- **Мощи** (reliquiae) → עֲצָמוֹת (ацамот) — кости; нет культа мощей в ТаНаХе
- **Облатка** (oblata) → לֶחֶם (лехем) — хлеб; нет аналога
- **Отлучение** (excommunicatio) → נִדּוּי (нидуй) — отстранение; но в ТаНаХе нет практики отлучения как в церкви
- **Папа** (papa) → нет аналога в ТаНаХе. Титул главы римского собрания
- **Паства** (pastor) → עֵדֶר (эдер) — стадо
- **Покаяние** (paenitentia) → תְּשׁוּבָה (тшува) — возвращение (но это не латинизм, а религионим. Здесь как пример чуждого понятия)
- **Понтифик** (pontifex) → нет аналога в ТаНаХе. Языческий титул римского императора, перешедший к папе
- **Пост** (ieiunium) → צוֹם (цом) — пост
- **Престол** (thronus) → כִּסֵּא (кисе) — сиденье, трон
- **Причастие** (communio) → שֻׁתָּפוּת (шутафут) — участие, общность
- **Проповедь** (praedicatio) → דְּרָשָׁה (драша) — истолкование; קְרִיאָה (криа) — чтение
- **Псалом** (psalmus) → מִזְמוֹר (мизмор) — песнь (грецизм, но закреплён латынью)
- **Распятие** (crucifixio) → צְלִיבָה (цлива) — повешение на столбе
- **Религия** (religio) → דֶּרֶךְ (дэрех) — путь
- **Ризница** (sacrarium) → לִשְׁכַּת הַכֵּלִים (лишкат hа-келим) — комната утвари
- **Святая вода** (aqua sancta) → מַיִם טְהוֹרִים (маим теhорим) — вода чистая (но без магии)
- **Святые дары** (sancta dona) → קָרְבָּן (корбан) — приближение
- **Святыня** (sanctuarium) → מִקְדָּשׁ (микдаш) — место отделённости
- **Священник** (sacerdos) → כֹּהֵן (коhэн) — служитель
- **Собор** (concilium) → קְהִלָּה (кеhила) — собрание
- **Соборование** (unctio) → מִשְׁחָה (мишха) — помазание; нет ритуала соборования
- **Сослужитель** (concelebrans) → עוֹבֵד (овед) — служащий вместе; нет аналога
- **Спасение** (salvatio) → יְשׁוּעָה (йешуа) — избавление
- **Страсть** (passio) → עִנּוּי (инуй) — страдание; нет теологии страстей
- **Схизма** (schisma) → פִּלּוּג (пилуг) — разделение, раскол
- **Таинство** (sacramentum) → מִצְוָה (мицва) — связь, соединение; או סוֹד (сод) — тайна (совет)
- **Тестаментум** (testamentum) → בְּרִית (брит) — союз
- **Уния** (unio) → אַחְדוּת (ахдут) — единство; не в смысле церковной унии
- **Целибат** (caelibatus) → нет аналога в ТаНаХе. Запрет на брак противоречит Торе
- **Церковь** (ecclesia) → קְהִלָּה (кеhила) — собрание
- **Чистилище** (purgatorium) → нет аналога в ТаНаХе. Понятие отсутствует
- **Экзорцизм** (exorcismus) → нет аналога в ТаНаХе. Изгнание нечистых дыханий как ритуал
- **Язычество** (paganus) → עֲבוֹדָה זָרָה (авода зара) — служение чужому; או גּוֹיִם (гоим) — народы
