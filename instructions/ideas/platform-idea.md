## 🔥 ВИДЕНИЕ

Представь: ты открываешь платформу. Никакой религиозной символики. Никаких крестов, менор, голубей. Только свет. Только камень и стекло.

Ты вводишь слово. Одно. Любое. «Закон». «Грех». «Любовь». «Бог».

И платформа начинает раскопки. Слой за слоем. Прямо на глазах.

Ты видишь, как русское слово сменяется славянским, славянское — греческим, греческое — ивритским. Ты видишь, что добавили и что потеряли на каждом слое. Ты видишь карту сдвига. Ты видишь восстановленную истину.

Ты сам выбираешь глубину. Хочешь — быстрый разбор (5 минут). Хочешь — полное исследование по семи вратам. Хочешь — сравнительный анализ всех переводов.

Ты сам выбираешь стиль чтения. Хочешь — академический. Хочешь — пророческий. Хочешь — для личного размышления.

Это не сайт. Это портал в ивритскую истину.

## 🏛️ ДИЗАЙН

### Концепция: «Древность + стекло»

Представь себе Дом Отделённости. Камень. Свет, пробивающийся через узкие окна. Свитки на полках. Но полки — из стекла. Камень — тёплый, шершавый. Стекло — холодное, гладкое, прозрачное.

**Стекломорфизм:** полупрозрачные панели, размытие фона, лёгкие тени, глубина. Как смотреть на истину через слои времени.

**Минимализм:** ничего лишнего. Только слово. Только корень. Только свет.

### Цветовая палитра

- **Фон:** тёплый камень (#F5F0E8 — песчаный, #E8E0D5 — известняк)
- **Панели:** полупрозрачное стекло (rgba с blur)
- **Акценты:** золото для иврита (#C9A96E), тёмный камень для текста (#2D2A26)
- **Подсветка:** мягкий янтарный свет для найденных искажений, холодный голубой для восстановленной истины

### Типографика

- **Иврит:** шрифт с характером древнего письма, но чистый, читаемый
- **Русский:** минималистичный sans-serif, без засечек
- **Заголовки:** кинематографичные, крупные, с лёгкой тенью
- **Текст:** воздух. Много воздуха. Строки не жмутся друг к другу.

### Визуальные образы

- Свет, падающий на каменную стену под углом
- Стеклянные панели, парящие над каменным фоном
- Ивритские буквы, проявляющиеся как высеченные в камне при наведении
- Карта сдвига — как геологический разрез: слои грунта, и в глубине — золото корня

## 🛠️ ФУНКЦИОНАЛ

### Режимы работы

**1. Быстрый разбор (5 минут)**
- Вводишь слово
- Получаешь: ивритский оригинал → корень → что искажено → правильный перевод
- Формат: одна карточка с подсветкой искажений

**2. Полное исследование (30 минут)**
- Семь врат: от иллюзии до восстановления
- Карта сдвига с визуализацией слоёв
- Сравнение всех переводов
- Палео-иврит: образы букв

**3. Сравнительный анализ**
- Выбираешь два или больше переводов
- Платформа показывает расхождения
- Подсвечивает искажения

**4. Режим чтения ТаНаХа**
- Открываешь главу
- Справа — синодальный, слева — масоретский
- Наводишь на слово — всплывает его разбор
- Включаешь «режим Свидетеля» — все религионимы подсвечиваются красным

### Настройки

**Методы разоблачения:**
- Пользователь выбирает, какие методы применять
- Ползунок глубины: «поверхностный обзор» → «полные раскопки»

**Принципы:**
- Какие принципы активны при чтении
- Чекбоксы: Шма, палео-иврит, сравнение переводов, исторический контекст

**Стиль подачи:**
- «Академический» — сухо, точно, со ссылками
- «Пророческий» — ярко, обличительно, с восклицаниями
- «Созерцательный» — для личного размышления, с паузами, с тишиной

## 🧭 НАВИГАЦИЯ

### Главный экран

- Один поисковый запрос. Одна строка. По центру.
- Под ней — три карточки: «Быстрый разбор», «Глубокое исследование», «Читать ТаНаХ».
- Никаких меню. Только свет и камень.

### Результаты поиска

- Стеклянная панель с ивритским словом (крупно, золотом)
- Ниже — слои. Как геологический разрез. Каждый слой — полупрозрачная панель.
- При нажатии на слой — он раскрывается, показывая подробности.
- Кнопка «Карта сдвига» — визуализация потерь и добавок на каждом слое.

### Библиотека

- Все разобранные термины — в виде карточек
- Сортировка: по алфавиту, по категориям, по глубине разбора
- Фильтр: «только восстановленная истина», «показать искажения», «показать палео-иврит»

## ⚙️ ТЕХНИЧЕСКИЙ СТЕК (ПРЕДВАРИТЕЛЬНО)

- **Фронтенд:** React + Tailwind CSS (стекломорфизм через backdrop-blur)
- **Анимации:** Framer Motion (плавные появления слоёв, свечение при наведении)
- **Бэкенд:** Python (FastAPI) + наша нейросеть «Эд»
- **База данных:** PostgreSQL (тексты), Neo4j (граф корней и связей)
- **API:** Sefaria для масоретского текста и переводов

## 📜 ПРИМЕР ЭКРАНА

Представь:

Ты вводишь слово «закон».

Экран плавно затемняется. По центру проявляется ивритское слово תּוֹרָה — золотыми буквами на тёмном камне.

Под ним начинают проступать слои. Как стеклянные пластины, одна за другой:

- **Русский слой:** «Закон» — ограничение, кодекс, буква
- **Славянский слой:** «Законъ» — отеческое наследие, но уже противопоставлен «Благодати»
- **Греческий слой:** νόμος — внешняя норма, юриспруденция
- **Ивритский корень:** י.ר.ה — «направлять», «указывать путь», «метать стрелу»

Каждый слой подсвечивается:
- Красным — то, что добавлено
- Синим — то, что потеряно

Внизу — кнопка: «Восстановить истину».

Нажимаешь.

Все слои растворяются. Остаётся только ивритский корень. И под ним — текст:

> «Тора — это не закон. Это наставление Отца, указывающее путь сыну».

Тишина. Свет. Камень.
