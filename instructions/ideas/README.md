## 📝 ПРАВИЛА

1. **Формат:** каждый файл с идеей должен содержать:
   - Название
   - Описание
   - Цель
   - План реализации
   - Приоритет

2. **Приоритизация:** указывать приоритет (high/medium/low)
3. **Статус:** обновлять статус (idea/planned/in-progress/completed)
4. **Ссылки:** ссылаться на связанные идеи
