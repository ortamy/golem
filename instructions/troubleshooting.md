## 📂 ПРОБЛЕМЫ С ФАЙЛАМИ

**ПРОБЛЕМА:** файл не сохраняется после копирования из чата

**РЕШЕНИЕ:**
- убедись, что скопирован весь блок кода
- проверь кодировку (должна быть UTF-8)
- удали скрытые символы форматирования

**ПРОБЛЕМА:** файл есть, но STRUCTURE.md его не видит

**РЕШЕНИЕ:**
```bash
python tools/sync/sync-structure.py
```

**ПРОБЛЕМА:** имя файла содержит русские буквы

**РЕШЕНИЕ:**
- переименовать в латиницу через дефис, КАПСОМ
- обновить STRUCTURE.md
- обновить ссылки на этот файл

---

## 🐍 ПРОБЛЕМЫ С PYTHON СКРИПТАМИ

**ПРОБЛЕМА:** `ModuleNotFoundError: No module named 'lib'`

**РЕШЕНИЕ:**
- проверь что запускаешь из правильной папки
- `sys.path` должен указывать на родительскую папку
- для utils/: `sys.path.insert(0, str(Path(__file__).parent.parent))`

**ПРОБЛЕМА:** `ModuleNotFoundError: No module named 'rich'`

**РЕШЕНИЕ:**
```bash
pip install rich
```

**ПРОБЛЕМА:** скрипт не находит файлы

**РЕШЕНИЕ:**
- запускай скрипт из корня проекта
- проверь пути: `REPO_ROOT = Path(__file__).parent.parent`
- для utils/: `REPO_ROOT = Path(__file__).parent.parent.parent`

**ПРОБЛЕМА:** `SyntaxError` при запуске

**РЕШЕНИЕ:**
- проверь версию Python: `python --version` (нужно 3.6+)
- проверь отступы (4 пробела, не табуляция)

## 🤖 ПРОБЛЕМЫ С CLINE

**ПРОБЛЕМА:** Cline не отвечает

**РЕШЕНИЕ:**
- проверь интернет
- проверь баланс DeepSeek API
- перезапусти VS Code

**ПРОБЛЕМА:** Error 429 — Daily free limit reached

**РЕШЕНИЕ:**
- ждать до следующего дня
- или пополнить баланс DeepSeek ($2-3)

**ПРОБЛЕМА:** Cline использует религионимы

**РЕШЕНИЕ:**
- загрузи `instructions/FORBIDDEN-WORDS.md`
- загрузи `instructions/tahor/`
- напомни в чате: «используй ивритские термины»

**ПРОБЛЕМА:** Cline удалил не то

**РЕШЕНИЕ:**
- откати через Git: `git checkout -- файл`
- или Ctrl+Z в редакторе

**ПРОБЛЕМА:** Checkpoints initialization timed out

**РЕШЕНИЕ:**
- нажать Disable checkpoints
- или открыть VS Code в папке с Git-репозиторием
