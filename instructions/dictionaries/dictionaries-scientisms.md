## 🔥 ВВЕДЕНИЕ

Наукоидолы — научные термины, которые система превратила в магические заклинания. Их используют не для познания, а для подавления. «Наука доказала» звучит как «так говорит Яхве». Но наука — не бог. Это инструмент познания творения, а не Творец.

## 📜 СЛОВАРЬ

### Физика и космология

- **Энергия** — способность совершать работу → כֹּחַ (коах) — сила. Подмена: сила Яхве заменена на безличную энергию
- **Квантовый** — относящийся к квантам → סוֹד (сод) — тайна. Подмена: тайна творения заменена на «квантовую магию»
- **Вибрации** — колебания → רַעַשׁ (рааш) — шум, колебание. Подмена: физическое явление стало «духовной энергией»
- **Частота** — число колебаний → מִסְפָּר (миспар) — число. Подмена: измерение заменено на мистику
- **Поле** (физическое) — область действия сил → שָׂדֶה (саде) — поле. Подмена: физическое поле заменено на «энергетическое поле»
- **Резонанс** — совпадение частот → הֵעָנוֹת (hеанут) — отклик. Подмена: физический отклик заменён на «духовный резонанс»
- **Гравитация** — притяжение масс → מָשַׁךְ (машах) — тянуть. Подмена: сила Яхве заменена на безличное притяжение
- **Большой взрыв** — теория происхождения вселенной → בְּרִיאָה (бриа) — творение. Подмена: творение Яхве заменено на случайный взрыв
- **Тёмная материя** — невидимая масса → תֹּהוּ (тоhу) — пустота. Подмена: непознанное названо «тёмной материей»
- **Чёрная дыра** — область сжатия → שְׁאוֹל (шеол) — могила, бездна. Подмена: шеол заменён на космический объект

### Биология и эволюция

- **Эволюция** — постепенное развитие → הִשְׁתַּנּוּת (hиштаннут) — изменение. Подмена: творение Яхве заменено на случайный процесс
- **Естественный отбор** — выживание приспособленных → בְּחִירָה (бхира) — выбор. Подмена: выбор Яхве заменён на слепой отбор
- **Ген** — единица наследственности → זֶרַע (зера) — семя. Подмена: семя Авраhама заменено на генетический код
- **Мутация** — изменение гена → מוּם (мум) — недостаток, порок. Подмена: порок заменён на «двигатель эволюции»
- **Инстинкт** — врождённое поведение → יֵצֶר (йецер) — помышление, склонность. Подмена: помышление сердца заменено на инстинкт
- **Популяция** — совокупность особей → עֵדֶר (эдер) — стадо. Подмена: творения Яхве заменены на «популяции»

### Химия и материя

- **Химия** — наука о веществах → אַלְכִּימְיָה (алкимия) — алхимия. Подмена: превращение веществ заменено на магию
- **Элемент** — простое вещество → יְסוֹד (йесод) — основание, стихия. Подмена: основание творения заменено на химический элемент
- **Реакция** — химическое превращение → תְּגוּבָה (тегува) — ответ, реакция. Подмена: действие Яхве заменено на химическую реакцию
- **Молекула** — частица вещества → אָבָק (авак) — пыль. Подмена: прах земной заменён на молекулу
- **Атом** — неделимая частица → גַּרְגִּיר (гаргир) — крупица. Подмена: мельчайшая частица творения заменена на атом

### Математика и информация

- **Информация** — сведения, данные → דַּעַת (даат) — знание. Подмена: живое знание заменено на мёртвую информацию
- **Код** — система знаков → סוֹד (сод) — тайна. Подмена: тайна заменена на программный код
- **Алгоритм** — последовательность действий → חֹק (хок) — устав. Подмена: устав Яхве заменён на алгоритм
- **Программа** — последовательность команд → תָּכְנִית (тохнит) — план, замысел. Подмена: замысел Яхве заменён на компьютерную программу
- **Матрица** — таблица чисел → תַּבְנִית (тавнит) — образец, чертёж. Подмена: образец творения заменён на математическую матрицу
- **Вычисление** — расчёт → חֶשְׁבּוֹן (хешбон) — счёт, отчёт. Подмена: отчёт перед Яхве заменён на вычисление

### Научный метод и авторитет

- **Наука** — система знаний → מַדָּע (мада) — наука, знание. ТаНаХ: «Знание надмевает, а любовь назидает» (1 Коринфянам 8:1)
- **Исследование** — научный поиск → דָּרַשׁ (дараш) — искать, исследовать. Подмена: исследование Торы заменено на научный поиск
- **Доказательство** — подтверждение → עֵדוּת (эдут) — свидетельство. Подмена: свидетельство очевидца заменено на научное доказательство
- **Факт** — установленная истина → אֱמֶת (эмет) — надёжность, истина. Подмена: надёжность Яхве заменена на научный факт
- **Теория** — научное объяснение → מָשָׁל (машал) — притча, сравнение. Подмена: притча заменена на теорию
- **Гипотеза** — предположение → הַשְׁעָרָה (hашара) — догадка, предположение
- **Эксперимент** — опыт → נִסָּיוֹן (нисайон) — испытание. Подмена: испытание веры заменено на научный эксперимент

### Медицина и тело (научный аспект)

- **Генетика** — наука о наследственности → תּוֹלְדוֹת (тольдот) — родословие. Подмена: родословие заменено на генетику
- **Клонирование** — создание копии → בָּרָא (бара) — творить. Подмена: творение Яхве заменено на клонирование
- **ГМО** — генетически модифицированный → כִּלְאַיִם (кильаим) — смешение видов. ТаНаХ: «Не смешивай двух видов» (Ваикра 19:19)
- **Стволовые клетки** — недифференцированные клетки → גֹּלֶם (голем) — неоформленная материя. Подмена: голем заменён на стволовую клетку
- **Вакцина** — препарат для иммунизации → תְּרוּפָה (теруфа) — лекарство. Подмена: лекарство заменено на вакцину как средство контроля

### Сознание и мозг (научный аспект)

- **Сознание** — осознание себя → נְשָׁמָה (нешама) — дыхание жизни. Подмена: дар Яхве заменён на функцию мозга
- **Подсознание** — скрытая часть психики → לֵב (лев) — сердце. Подмена: сердце как центр воли заменено на подсознание
- **Нейрон** — нервная клетка → עָצָב (ацав) — нерв. Подмена: живая ткань заменена на «нейронную сеть»
- **Сигнал** — электрический импульс → רוּחַ (руах) — дыхание, ветер. Подмена: дыхание жизни заменено на электрический сигнал
