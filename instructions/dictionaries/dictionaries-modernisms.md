## 🔥 ВВЕДЕНИЕ

Модернизмы — пустые или искажённые слова, вошедшие в язык через современную культуру, молодёжный сленг, интернет и масс-медиа. Они не имеют корней ни в ТаНаХе, ни в классическом языке. Они созданы чтобы заменить живые понятия на пустые оболочки.

## 📜 СЛОВАРЬ

### Сленговые и интернет-слова

- **Кринж** (cringe) — съёживаться от стыда → בּוּשָׁה (буша) — стыд, позор. Или: יִרְאָה (йиръа) — трепет
- **Краш** (crush) — объект влюблённости → אַהֲבָה (аhава) — любовь-действие. Или: תַּאֲוָה (таава) — вожделение
- **Токсик** (toxic) — ядовитый, вредный → רַע (ра) — злой, разрушительный. Или: טָמֵא (тамэ) — непригодный
- **Абьюз** (abuse) — насилие, жестокое обращение → חָמָס (хамас) — насилие. Или: עֹשֶׁק (ошек) — угнетение
- **Газлайтинг** — манипулятивное обесценивание → שֶׁקֶר (шекер) — ложь. Или: רְמִיָּה (ремия) — обман
- **Хейт** (hate) — ненависть → שִׂנְאָה (синъа) — ненависть
- **Хайп** (hype) — шумиха, ажиотаж → הֶבֶל (hевель) — суета, дуновение
- **Флекс** (flex) — хвастовство, показуха → גַּאֲוָה (гаава) — гордость. Или: הִתְפָּאֲרוּת (hитпаарут) — хвастовство
- **Рофл** (ROFL) — громко смеяться → שְׂחוֹק (схок) — смех. Или: לַעַג (лааг) — насмешка
- **Тролль** — провокатор → לֵץ (лец) — насмешник, глумливый
- **Чилл** (chill) — расслабление → שָׁבַת (шават) — прекращать, покоиться. Или: מְנוּחָה (менуха) — покой
- **Вайб** (vibe) — атмосфера, ощущение → רוּחַ (руах) — дыхание, дух, атмосфера
- **Пруф** (proof) — доказательство → עֵדוּת (эдут) — свидетельство. Или: אוֹת (от) — знак
- **Кринжовый** — вызывающий стыд → מֵבִישׁ (мевиш) — постыдный
- **Зашквар** — позор, нечто неприемлемое → תּוֹעֵבָה (тоэва) — мерзость
- **ЛП** (лучшая подруга) → רֵעָה (реа) — подруга, ближняя
- **ЛЧ** (лучший человек) → אָהוּב (аhув) — возлюбленный
- **Тян** (от яп. -тян) — девушка → נַעֲרָה (наара) — девушка. Или: בְּתוּלָה (бетула) — девица
- **Кун** (от яп. -кун) — парень → נַעַר (наар) — юноша
- **Ачивка** (achievement) — достижение → מַעֲשֶׂה (маасэ) — дело, действие
- **Фармиться** (farm) — накапливать, собирать → אָסַף (асаф) — собирать

### Масс-медиа и реклама

- **Контент** — содержимое → תֹּכֶן (тохен) — содержание. Или: דָּבָר (давар) — слово, дело
- **Платформа** — площадка → מָקוֹם (маком) — место
- **Блогер** — ведущий дневник → מְסַפֵּר (месапер) — рассказчик
- **Инфлюенсер** — влияющий → מַשְׁפִּיעַ (машпиа) — влияющий. Или: מֵסִית (месит) — подстрекатель
- **Подписчик** — последователь → תַּלְמִיד (талмид) — ученик. Или: עוֹקֵב (окев) — следующий
- **Лайк** — одобрение → אָמֵן (амен) — подтверждение. Или: רָצוֹן (рацон) — благоволение
- **Дизлайк** — неодобрение → תּוֹכֵחָה (тохеха) — обличение
- **Репост** — пересылка → שָׁלַח (шалах) — послать
- **Сторис** — история на сутки → חָזוֹן (хазон) — видение. Или: רֶגַע (рега) — мгновение
- **Мем** — смешная картинка → מָשָׁל (машал) — притча. Или: לַעַג (лааг) — насмешка
- **Тренд** — течение, мода → זֶרֶם (зерем) — поток, течение
- **Челлендж** — вызов → נִסָּיוֹן (нисайон) — испытание
- **Свайп** — скольжение пальцем → הָפַךְ (hафах) — перевернуть
- **Скролл** — прокрутка → גָּלַל (галал) — катить, сворачивать
- **Бан** — запрет → חֵרֶם (херем) — отлучение
- **Мьют** — заглушить → אָטַם (атам) — затыкать, закрывать

### Абстрактные и пустые

- **Самореализация** → нет аналога в ТаНаХе. Ближайшее: עֲבוֹדַת יְהוָה (аводат Яхве) — служение Яхве
- **Личностный рост** → בִּינָה (бина) — разумение. Или: חָכְמָה (хохма) — мудрость
- **Зона комфорта** → מִדְבָּר (мидбар) — пустыня. Или: אֶרֶץ (эрец) — земля
- **Позитив** → טוֹב (тов) — пригодное, хорошее
- **Негатив** → רַע (ра) — разрушительное
- **Энергетика** → רוּחַ (руах) — дыхание, дух
- **Космос** (в эзотерическом смысле) → שָׁמַיִם (шамаим) — небеса
- **Вселенная** (как божество) → עוֹלָם (олам) — скрытый горизонт
- **Карма** → גְּמוּל (гемуль) — воздаяние
- **Аура** → אוֹר (ор) — свет. Или: כָּבוֹד (кавод) — слава, тяжесть
- **Чакры** → нет аналога. Ближайшее: אֵיבָרִים (эварим) — члены тела
- **Медитация** → הָגָה (hага) — размышлять, шептать
- **Аффирмация** → בְּרָכָה (браха) — благословение. Или: אָמַר (амар) — сказать
- **Визуализация** → חָזָה (хаза) — видеть, прозревать
- **Манифестация** → בָּרָא (бара) — творить. Или: עָשָׂה (аса) — делать

### Отношения и психология

- **Партнёр** (в отношениях) → אִישׁ (иш) — муж. Или: אִשָּׁה (иша) — жена
- **Бывший/бывшая** → גָּרוּשׁ (гаруш) — разведённый
- **Созвон** → קָרָא (кара) — звать, призывать
- **Чатиться** → דִּבֵּר (дибер) — говорить
- **Гуглить** → דָּרַשׁ (дараш) — искать, исследовать
- **Загуглить** → בִּקֵּשׁ (бикеш) — просить, искать
- **Сёрфить** (в интернете) → תָּעָה (таа) — блуждать
- **Зависать** → יָשַׁב (яшав) — сидеть, пребывать
- **Тусить** → חָגַג (хагаг) — праздновать
- **Задонатить** → נָתַן (натан) — дать. Или: נְדָבָה (недава) — пожертвование
- **Буллинг** — травля → רָדַף (радаф) — преследовать
- **Сталкинг** — преследование → עָקַב (акав) — следить
- **Харассмент** — домогательство → תָּקַף (такаф) — нападать, принуждать
