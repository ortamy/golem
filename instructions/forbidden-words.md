## 📜 КАТЕГОРИИ

### Религионимы
Слова, подменяющие ивритские понятия на христианские или раввинистические.
- Ад → שְׁאוֹל (шеол)
- Ангел → מַלְאָךְ (малъах)
- Бог → אֱלֹהִים (Элоhим) / יְהוָה (Яхве)
- Вера → אֱמוּנָה (эмуна)
- Господь → יְהוָה (Яхве)
- Грех → חֵטְא (хэт)
- Дух → רוּחַ (руах)
- Духовность → רוּחָנִיּוּת (руханийут) — относящееся к дыханию
- Душа → נֶפֶשׁ (нэфеш)
- Молитва → תְּפִלָּה (тфила)
- Пророк → נָבִיא (нави)
- Религиозный → דָּתִי (дати) — соблюдающий; или שַׁיָּךְ לַדֶּרֶךְ (шая́х ла-дэ́рэх) — принадлежащий пути
- Религия → דֶּרֶךְ (дэрех)
- Святой → קֹדֶשׁ (кодеш)
- Спасение → יְשׁוּעָה (йешуа)
- Храм → בַּיִת (Дом) / מִקְדָּשׁ (микдаш)
- Церковь → קְהִלָּה (кеhила)
- ...и ещё 30+ слов

## ⚖️ ПРАВИЛА ПРИМЕНЕНИЯ

1. **Авторская речь:** замена обязательна.
2. **Контекст разоблачения:** допустимо использовать запрещённое слово в кавычках или с пометкой «синодальное».
3. **Первый раз в тексте:** иврит + транслитерация + перевод. Затем — только выбранное слово.
4. **Карта пополняется.** Каждое новое найденное искажение добавляется в соответствующий файл категории.
