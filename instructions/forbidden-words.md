## 📜 КАТЕГОРИИ

### Религионимы
Слова, подменяющие ивритские понятия на христианские или раввинистические.
- Ад → שְׁאוֹל (шеол)
- Ангел → מַלְאָךְ (малъах)
- Бог → אֱלֹהִים (Элоhим) / יְהוָה (Яхве)
- Вера → אֱמוּנָה (эмуна)
- Господь → יְהוָה (Яхве)
- Грех → חֵטְא (хэт)
- Дух → רוּחַ (руах)
- Духовность → רוּחָנִיּוּת (руханийут) — относящееся к дыханию
- Душа → נֶפֶשׁ (нэфеш)
- Молитва → תְּפִלָּה (тфила)
- Пророк → נָבִיא (нави)
- Религиозный → דָּתִי (дати) — соблюдающий
- Религия → דֶּרֶךְ (дэрех)
- Святой → קֹדֶשׁ (кодеш)
- Спасение → יְשׁוּעָה (йешуа)
- Храм → בַּיִת (Дом) / מִקְדָּשׁ (микдаш)
- Церковь → קְהִלָּה (кеhила)

📁 Полный список (30+): `instructions/tahor/religionims.md`

### Грецизмы
Слова, пришедшие из греческого и заменившие ивритские понятия.
- Апостол → שָׁלִיחַ (шалиах)
- Евангелие → בְּשׂוֹרָה (бсора)
- Идеология → מַחְשָׁבָה (махшава)
- Крещение → טְבִילָה (твила)
- Христос → מָשִׁיחַ (Машиах)

📁 Полный список (25+): `instructions/tahor/grecisms.md`

### Церковнославянизмы
Слова, созданные или переосмысленные в церковнославянском.
- Блуд → זָנָה (зана)
- Воскресение → קִימָה (кима)
- Православие → אֱמוּנָה נְכוֹנָה (эмуна нехона)
- Прелюбодеяние → נָאַף (нааф)
- Троица → нет аналога в ТаНаХе

📁 Полный список (20+): `instructions/tahor/slavicisms.md`

### Латинизмы
Слова, пришедшие из латыни и оставленные без перевода.
- Индульгенция → нет аналога
- Религия → דֶּרֶךְ (дэрех)
- Языческий → שַׁיָּךְ לָעַמִּים (шая́х ла-ами́м)
- Язычество → עֲבוֹדָה זָרָה (авода зара)

📁 Полный список (5+): `instructions/tahor/latinisms.md`

### Кастрированные слова
Слова, сохранившие оболочку но потерявшие смысл.
- Физика → φύσις (фюсис) — рождение сущего
- Математика → μάθημα (матема) — познание
- Технология → τέχνη + λόγος (техне + логос) — искусство воплощения смысла
- Пост → צוֹם (цом) — помощь другому, а не ритуальный голод
- Духовность → слово-пустышка, нет аналога
- Самореализация → слово-пустышка, нет аналога

📁 Полный список: `instructions/exposure/exposure-language.md`

### Научные термины-подмены
Научные слова, используемые как магические понятия.
- Энергия → физический термин, не магическая сила
- Вибрации → физическое явление, не «духовные вибрации»
- Квантовый → раздел физики, не маркетинговый термин

### Имена и названия
Собственные имена и географические названия, искажённые в переводах.
- Иерусалим → יְרוּשָׁלַיִם (Йерушалаим)
- Иисус → יֵשׁוּעַ (Йешуа)
- Моисей → מֹשֶׁה (Моше)
- Пасха → פֶּסַח (Песах)

📁 Полный список (20+): `instructions/tahor/names.md`

### Фразы и выражения
Устойчивые обороты, требующие возврата к ивритскому смыслу.
- «Страх Божий» → יִרְאַת יְהוָה (йиръат Яхве)
- «Святой Дух» → רוּחַ הַקֹּדֶשׁ (Руах hа-Кодеш)
- «Вечный завет» → בְּרִית עוֹלָם (брит олам)

📁 Полный список (10+): `instructions/tahor/phrases.md`
