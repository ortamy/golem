## 🎯 ТРИ МЕХАНИЗМА ПОДМЕНЫ

### 1. ЗАТУМАНИВАНИЕ ЯСНОГО РАЗЛИЧИЯ
טָהוֹר и טָמֵא → «чистое» и «нечистое». Функция стала моралью.

### 2. СГЛАЖИВАНИЕ ОСТРЫХ УГЛОВ ОТКРОВЕНИЯ
יִרְאָה (видение/трепет) → «страх». Откровение стало угрозой.

### 3. ПОДМЕНА МНОГОМЕРНОЙ ИСТИНЫ ПЛОСКИМ СУРРОГАТОМ
אֱמוּנָה (верность) → «вера». Отношения стали идеологией.

## 🎯 ТИПИЧНЫЕ ПРИЁМЫ ПОДМЕНЫ

### ЯЗЫКОВЫЕ ПРИЁМЫ
- **Транслитерация вместо перевода:** διάβολος → «дьявол»
- **Замена конкретного абстрактным:** נֶפֶשׁ → «душа»
- **Замена действия состоянием:** אֱמוּנָה → «вера»
- **Замена функции титулом:** ἐπίσκοπος → «епископ»
- **Замена описания именем:** διάβολος → «Дьявол»
- **Смешение нескольких слов в одно:** אַהֲבָה, רָחַם, חֶסֶד → «любовь»
- **Разрыв корневых связей:** צַדִּיק, צְדָקָה, הִצְדִּיק → три разных слова
- **Смещение времени:** пророчество → история
- **Скрытие игры слов:** אוֹר и עוֹר — омофоны
- **Подмена языка творения на язык описания:** דָּבָר → λόγος
- **Подмена физического на метафорическое:** помазание маслом → «помазание духом»
- **Подмена отсутствия на метафору:** нет помазания → «духовное помазание»

### СМЫСЛОВЫЕ ПРИЁМЫ
- **Подмена порядка на иерархию:** «у всего есть место» → «это выше»
- **Подмена функционального на моральное:** «непригодное» → «плохое»
- **Подмена прозрачности на стыд:** нагота → стыд
- **Замена звука на образ:** голос Яхве → картинка
- **Подмена присутствия на здание:** מִקְדָּשׁ → «храм»
- **Подмена близости на дистанцию:** Имя → обычай не произносить
- **Подмена близости на угрозу:** видение → «страх»
- **Подмена Имени на титул:** יהוה → «Господь»
- **Подмена центра мышления:** сердце → мозг
- **Подмена исцеления на лечение:** целостность → устранение симптомов
- **Подмена погребения на помазание:** миро → елей
- **Подмена через исполнение:** не отменить, а «исполнить»
- **Подмена формы без изменения содержания:** греческая оболочка, славянское нутро
- **Подмена мудрости на знания:** хохма → информация
- **Подмена дара на долг:** шмита → кредит
- **Подмена реального на виртуальное:** коллективное → атомизированное

### СОЦИАЛЬНЫЕ ПРИЁМЫ
- **Подмена действия на идентичность:** «служит чужому» → «язычник»
- **Подмена состояния на национальность:** עִבְרִי → «еврей»
- **Подмена имени на роль:** «как тебя зовут?» → «сколько ты сделал?»
- **Подмена защиты на осуждение:** заповедь «не убей» → обвинение жертвы
- **Подмена состояния на приговор:** «ты в состоянии» → «ты приговорён»
- **Подмена свободы уйти на приговор:** свобода → проклятие
- **Подмена призвания на мощь:** хранитель сада → разрушитель
- **Подмена служения на власть:** царь-слуга → царь-господин
- **Подмена договора на государство:** доверие → надсмотрщик
- **Подмена остатка на большинство:** «все так делают» → значит истина
- **Подмена коллективного на атомизированное:** вместе → поодиночке

### ЭКОНОМИЧЕСКИЕ ПРИЁМЫ
- **Подмена источника на цистерну:** живая вода → разбитый водоём
- **Подмена свободы на накопление:** «накопи — и будешь свободен»
- **Подмена веса на цифру:** שֶׁקֶל → валюта
- **Подмена ограничения на проклятие:** защита → угроза

### МЕТА-ПРИЁМЫ
- **Создание «отделённого» языка:** понятные слова → «богословские термины»
- **Подмена пути на систему:** דֶּרֶךְ → religio
- **Дуализация:** единое → противоположное
