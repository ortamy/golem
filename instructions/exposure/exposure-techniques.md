## 🔥 ВВЕДЕНИЕ

Приёмы — это конкретные техники, которыми система искажает истину. Механизмы (в отдельном файле) — глобальные принципы. Приёмы — инструменты. Знать приёмы — значит видеть руку системы в каждом искажении.

## 🗣 ЯЗЫКОВЫЕ ПРИЁМЫ

- **Транслитерация вместо перевода:** διάβολος → «дьявол»
- **Замена конкретного абстрактным:** נֶפֶשׁ → «душа»
- **Замена действия состоянием:** אֱמוּנָה → «вера»
- **Замена функции титулом:** ἐπίσκοπος → «епископ»
- **Замена описания именем:** διάβολος → «Дьявол»
- **Смешение нескольких слов в одно:** אַהֲבָה, רָחַם, חֶסֶד → «любовь»
- **Разрыв корневых связей:** צַדִּיק, צְדָקָה, הִצְדִּיק → три разных слова
- **Смещение времени:** пророчество → история
- **Скрытие игры слов:** אוֹר и עוֹר — омофоны
- **Подмена языка творения на язык описания:** דָּבָר → λόγος
- **Подмена физического на метафорическое:** помазание маслом → «помазание духом»
- **Подмена отсутствия на метафору:** нет помазания → «духовное помазание»
- **Создание «отделённого» жаргона:** понятные слова → «богословские термины»
- **Этимологическая подмена:** «выбирающий» (αἱρετικός) → «еретик» (преступник)
- **Сокрытие смысла в заимствовании:** ὑποθήκη (под-положенное) → «ипотека» (нейтрально)
- **Языковая честность английского против русской маскировки:** mortgage («мёртвый залог») vs «ипотека» (скрыто)
- **Подмена живого слова мёртвым термином:** наставление → «доктрина»
- **Подмена самоназвания ярлыком:** עִבְרִי (перешедший) → «еврей» (этнический ярлык)
