## 🔍 ВЫЯВЛЕННЫЕ ПРОБЛЕМЫ

### 1. Дублирование контента
- **`content/`** (корень) vs **`content/`** в подпапках — возможно, дублирование
- **`webapp/`** содержит те же языковые папки (ru/, en/, he/) — дублирует структуру
- **`tanakh/`** в корне содержит данные, но также есть **`ru/tanakh/`**, **`en/tanakh/`**, **`he/tanakh/`** — пересечение

### 2. Временные/технические файлы в корне
- **`fix_web_part3.js`** — временный скрипт для исправлений
- **`STRUCTURE.md`** — документация, должна быть в `docs/`
- **`files.json`** — неизвестное назначение, возможно не используется

### 3. Конфигурационные файлы в корне
- **`package.json`**, **`package-lock.json`** — NPM конфигурация
- **`postcss.config.js`**, **`purgecss.config.js`**, **`tailwind.config.js`** — конфигурация сборки
- **`server.js`** — dev сервер
- **`app.js`** — вспомогательный скрипт

### 4. Пустые/незаполненные папки
- **`lang/`** — содержит папки для 8 языков (ar, de, es, fr, ja, ko, pt, zh), но нет файлов
- **`export/`** — содержит 2 HTML файла, непонятно используются ли они

### 5. Неясная структура
- **`webapp/`** — дублирует языковую структуру, но в другом формате
- **`tanakh/`** — содержит тестовые файлы и данные, смешивается с контентом
- **`src/`** — исходный CSS, но есть также `style.css` в корне

### 6. Отсутствие стандартных папок
- Нет **`assets/`** для изображений, шрифтов, медиа
- Нет **`docs/`** для документации
- Нет **`build/`** для конфигурации сборки
