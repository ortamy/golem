## web/icons/32/ — Категории контента (32×32 px)

Используются в заголовках H1, на страницах категорий, в логотипе.

- `scroll.svg` — развёрнутый свиток с двумя роликами. Категория: Терминология
- `tablets.svg` — две каменные скрижали. Категория: ТаНаХ
- `fish.svg` — рыба (символ ихтис). Категория: БаШаХ
- `lamp.svg` — масляная лампа с пламенем. Категория: Исследования
- `tree.svg` — дерево с плодами и корнями. Категория: Учения
- `aleph.svg` — буква א (алеф). Категория: Изучение иврита
- `temple.svg` — две колонны с перекладиной. Категория: Храм / Архитектура

## web/icons/16/ — Интерфейс и действия (16×16 px)

Используются в кнопках, меню, сайдбаре, списке файлов.

- `search.svg` — лупа. Действие: поиск
- `menu.svg` — три горизонтальные линии. Действие: меню (бургер)
- `theme.svg` — солнце с лучами. Действие: переключение темы
- `copy.svg` — два наложенных листа. Действие: копировать ссылку
- `print.svg` — принтер. Действие: печать
- `random.svg` — два игральных кубика. Действие: случайный файл
- `bookmark.svg` — пустая звезда. Состояние: не в закладках
- `bookmark-active.svg` — заполненная звезда. Состояние: в закладках
- `close.svg` — крестик (X). Действие: закрыть
- `back.svg` — стрелка влево. Действие: назад
- `top.svg` — стрелка вверх. Действие: наверх
- `file.svg` — документ с загнутым углом. Элемент: файл
- `folder.svg` — папка. Элемент: категория
- `external.svg` — стрелка вверх-вправо. Элемент: внешняя ссылка
- `dot.svg` — кружок. Элемент: разделитель, индикатор
- `spinner.svg` — круг с разрывом. Состояние: загрузка
