◈ Инструменты проекта «Голем»

В этом разделе собраны инструменты для исследования ТаНаХа, анализа текстов и создания визуальных материалов. Каждый инструмент разработан для восстановления истины на основе ивритского оригинала.

Генератор исследовательских досок
Создавайте визуальные исследовательские доски для анализа улик, выводов и вложений. Экспортируйте в PNG, PDF или TXT. Генерируйте промпты для нейросетей.

Чекер религионимов (скоро)
Автоматическая проверка текстов на подмены, искажения и переводческие ошибки. Сравнение оригиналов с переводами.
