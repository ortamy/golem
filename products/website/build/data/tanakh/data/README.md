# 🗄️ ПАПКА DATA

**Описание:** Данные проекта «ГОЛЕМ»

---

## 📂 СТРУКТУРА

### JSON файлы

#### lxx.json - 📜 Септуагинта
Данные Септуагинты (древний перевод Танаха на греческий)

**Формат:** JSON  
**Назначение:** Хранение текста Септуагинты для сравнения с еврейским текстом

---

#### qumran.json - 📜 Мёртвое море
Данные рукописей Мёртвого моря

**Формат:** JSON  
**Назначение:** Хранение данных рукописей Кумрана для исследований

---

#### tanakh-books.json - 📚 Книги Танаха
Метаданные книг Танаха

**Формат:** JSON  
**Назначение:** Структурированные данные о книгах Танаха (автор, дата, категория)

---

#### translation.json - 🌍 Переводы
Данные переводов

**Формат:** JSON  
**Назначение:** Хранение переводов текстов для сравнения

---

### Подпапки

#### tanakh-cache/ - 💾 Кэш Танаха
Кэшированные данные Танаха

**Назначение:** Ускорение доступа к часто используемым данным Танаха

**Примечание:** Автоматически обновляется скриптами

## 🔗 СВЯЗИ

### Внешние связи
- `content/tanakh/` - контент Танаха
- `content/bashah/` - контент Брита Хадаша
- `tools/` - инструменты для работы с данными
- `products/` - продукты, использующие данные

---

## 📝 ПРАВИЛА

1. **Формат:** все данные в JSON формате
2. **Кодировка:** UTF-8
3. **Структура:** единый формат для всех JSON файлов
4. **Версионирование:** изменения фиксировать в CHANGELOG.md
5. **Кэш:** кэш обновляется автоматически, не редактировать вручную

---

## 🛠 ИСПОЛЬЗОВАНИЕ

### Чтение данных
```python
import json

# Чтение книг Танаха
with open('data/tanakh-books.json', 'r', encoding='utf-8') as f:
    books = json.load(f)

# Чтение Септуагинты
with open('data/lxx.json', 'r', encoding='utf-8') as f:
    lxx = json.load(f)
```

### Обновление кэша
```bash
# Обновление кэша Танаха
python tools/sync/update-tanakh-cache.py
```
