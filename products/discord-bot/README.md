# 🤖 Бот «Голем» — Поиск корней иврита

Discord-бот для поиска палео-ивритских корней из ТаНаХа.  
Отвечает на команду `/root` — находит корень слова, его смысл и пример из ТаНаХа.

---

## 📂 Структура

```
products/discord-bot/
├── bot.py          # основной код бота
├── roots.json      # база корней (150+ корней)
├── .env            # токен бота (создаётся вручную)
├── requirements.txt # зависимости
└── README.md       # инструкция
```

## 🛠 Установка

### 1. Клонировать репозиторий

```bash
git clone https://github.com/ortamy/golem.git
cd golem/products/discord-bot
```

### 2. Установить зависимости

```bash
pip install -r requirements.txt
```

### 3. Создать файл `.env`

Создайте файл `.env` в папке `products/discord-bot/` со следующим содержимым:

```
DISCORD_TOKEN=ваш_токен_бота
```

**Как получить токен:**
1. Перейдите на [Discord Developer Portal](https://discord.com/developers/applications)
2. Создайте новое приложение → Bot → Copy Token
3. Вставьте токен в файл `.env`

> ⚠️ **Никогда не коммитьте `.env` в Git!** Он добавлен в `.gitignore`.

### 4. Пригласить бота на сервер

В Developer Portal → OAuth2 → URL Generator:
- Scopes: `bot`, `applications.commands`
- Bot Permissions: `Send Messages`, `Embed Links`, `Read Message History`
- Откройте сгенерированный URL и пригласите бота на сервер

## 🚀 Запуск

```bash
python bot.py
```

Бот подключится к Discord и синхронизирует слэш-команды.  
Статус бота: **ТаНаХ | /root**

---

## 📖 Команды

| Команда | Описание |
|---------|----------|
| `/root <слово>` | Найти корень ивритского слова |
| `/about` | Информация о боте |

### Пример использования

```
/root אהב
```

Ответ:
- **Корень:** א.ה.ב
- **Смысл:** Любить, выбирать, быть верным
- **Пример:** וְאָהַבְתָּ אֵת יְהוָה אֱלֹהֶיךָ — И возлюби Яхве Всесильного твоего (Втор. 6:5)

---

## 🗂 База корней

Файл `roots.json` содержит **150+ корней** из ТаНаХа в формате:

```json
{
  "אהב": {
    "root": "א.ה.ב",
    "meaning": "Любить, выбирать, быть верным",
    "example": "וְאָהַבְתָּ אֵת יְהוָה אֱלֹהֶיךָ — И возлюби Яхве Всесильного твоего (Втор. 6:5)"
  }
}
```

Каждая запись содержит:
- **root** — трёхбуквенный корень (с точками между буквами)
- **meaning** — смысл корня на русском
- **example** — пример из ТаНаХа (оригинал + перевод + ссылка)
