Вспомогательный ассистент

### davar/ - 📝 Язык Давар
Язык программирования Давар

**Подпапки:**
- `examples/` - примеры кода
- `davar-architecture.md` - архитектура языка
- `davar-language.md` - спецификация языка

**Назначение:** Собственный язык программирования для проекта

---

### neuro/ - 🧠 Нейросеть Эд
Нейросеть для обработки естественного языка

**Подпапки:**
- `docs/` - документация
- `eval/` - оценка модели
- `inference/` - сервер и клиент
- `models/` - модели
- `scripts/` - скрипты
- `training-data/` - данные для обучения

**Назначение:** Нейросеть для анализа и генерации текста

Веб-приложение

## 🔗 СВЯЗИ

### Внешние связи
- `content/` - контент для продуктов
- `tools/` - инструменты для сборки
- `data/` - данные для продуктов

---

## 📝 ПРАВИЛА

1. **Изоляция:** каждый продукт изолирован в своей папке
2. **Зависимости:** зависимости указывать в `requirements.txt` внутри папки продукта
3. **Деплой:** инструкции по деплою в `docs/DEPLOY.md`
4. **Тестирование:** тесты в папке `tests/` внутри продукта

---
