# 🛠 ПАПКА TOOLS

**Описание:** Инструменты для работы с проектом «ГОЛЕМ»

---

## 📂 СТРУКТУРА

### Главные скрипты
- `golem.py` - главное меню проекта
- `audit-content.py` - аудит контента
- `fetch-sefaria.py` - загрузка данных с Sefaria
- `interlinear-generator.py` - генератор интерлинеарного текста
- `replace_emoji_with_icons.py` - замена эмодзи на иконки
- `replace_h2_emoji.py` - замена эмодзи в заголовках

### Подпапки

#### checkers/ - ✅ Проверки
Скрипты для проверки контента и структуры

**Назначение:** Автоматические проверки качества контента

#### generators/ - 🔨 Генераторы
Генераторы файлов и контента

**Назначение:** Автоматическая генерация файлов по шаблонам

#### reports/ - 📊 Отчёты
Генераторы отчётов

**Назначение:** Создание отчётов о состоянии проекта

#### automation/ - ⚡ Автоматизация
Скрипты автоматизации

**Назначение:** Автоматизация рутинных задач

#### backup/ - 💾 Бэкап
Скрипты резервного копирования

**Назначение:** Создание резервных копий проекта

#### lib/ - 📚 Общие утилиты
Общие библиотеки и утилиты

**Назначение:** Переиспользуемый код для всех скриптов

#### utils/ - 🔧 Вспомогательные утилиты
Дополнительные утилиты

**Назначение:** Вспомогательные функции

#### data/ - 📊 Данные
Данные для инструментов

**Назначение:** Вспомогательные данные для скриптов

#### sync/ - 🔄 Синхронизация
Скрипты синхронизации

**Назначение:** Синхронизация контента и структуры

## 📝 ПРАВИЛА

1. **Автоматическое обнаружение:** новые скрипты в `checkers/`, `generators/`, `reports/`, `automation/` автоматически появляются в меню `golem.py`
2. **Именование:** snake_case для имён файлов
3. **Документация:** каждый скрипт должен иметь docstring
4. **Тестирование:** тесты в папке `tests/` рядом со скриптом

## 🛠 ИСПОЛЬЗОВАНИЕ

### Главное меню
```bash
python tools/golem.py
```

### Отдельные скрипты
```bash
# Проверка контента
python tools/checkers/check-naming.py
python tools/checkers/validate-metadata.py
python tools/checkers/check-religionisms.py

# Генерация отчётов
python tools/reports/generate-stats.py

# Автоматизация
python tools/automation/sync-structure.py
```
