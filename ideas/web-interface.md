## 🔥 ЧТО ЭТО

Веб-интерфейс — это сайт, который предоставляет доступ к функционалу проекта «Голем» через браузер.

**Зачем нужен:**
- доступ без установки инструментов
- удобный поиск по репозиторию
- визуализация исследований
- онлайн-проверка текстов через чекеры
- мобильная версия

## 🎯 ФУНКЦИОНАЛ

**1. ГЛАВНАЯ СТРАНИЦА**
- приветствие и описание проекта
- последние добавленные исследования
- статистика (сколько терминов, исследований)
- быстрый доступ к основным разделам

**2. ТЕРМИНОЛОГИЯ**
- алфавитный список всех терминов
- поиск по названию и содержанию
- фильтр по категориям
- карточка термина: эмоджи, название, определение, ссылка на полный разбор

**3. ИССЛЕДОВАНИЯ**
- список всех исследований
- поиск по заголовкам и тегам
- сортировка по дате, названию
- полнотекстовый просмотр

**4. ОНЛАЙН-ЧЕКЕР**
- текстовое поле для ввода текста
- выбор типа проверки (bdikah, mivdak, factcheck)
- кнопка «Проверить»
- вывод результатов с подсветкой ошибок
- предложение замен для религионимов

**5. ЧТЕНИЕ ТАНАХА**
- выбор книги, главы, стиха
- отображение: иврит + транслитерация + перевод
- поиск по ключевым словам
- закладки

**6. ПОИСК**
- глобальный поиск по всем файлам
- подсветка найденных слов
- фильтр по папкам

**7. API (для разработчиков)**
- REST API для всех функций чекеров
- возможность интеграции в другие проекты

## 🛠️ ТЕХНИЧЕСКАЯ РЕАЛИЗАЦИЯ

**ФРОНТЕНД:**
- HTML5 + CSS3 (или Tailwind)
- JavaScript (ванильный или React/Vue)
- адаптивный дизайн

**БЭКЕНД (ОПЦИОНАЛЬНО):**
- Python + Flask/FastAPI
- или Node.js + Express
- кеширование результатов

**БАЗА ДАННЫХ (ОПЦИОНАЛЬНО):**
- SQLite (легко, без сервера)
- или PostgreSQL (для масштабирования)

**ДЕПЛОЙ:**
- статический сайт (GitHub Pages)
- или Vercel/Netlify
- или собственный сервер

---

## 🔄 ВЗАИМОДЕЙСТВИЕ С ИНСТРУМЕНТАМИ

**НА БЭКЕНДЕ:**
- вызов существующих чекеров через subprocess
- загрузка данных из репозитория (md-файлов)
- парсинг метаданных

**НА ФРОНТЕНДЕ:**
- чтение экспортированного JSON
- или AJAX-запросы к API

**ЭКСПОРТ ДАННЫХ ДЛЯ ВЕБА:**
```bash
python3 tools/export-to-json.py
# создаёт data.json для сайта
```
