## 🔥 НАВИГАЦИЯ И СТРУКТУРА

- Хлебные крошки — путь к файлу: «Главная > Учения > Католицизм»
- Раскрывающееся дерево категорий в сайдбаре (как папки)
- Быстрые фильтры: «Только красные», «Только зелёные», «Новые», «Популярные»
- Связанные файлы с preview — карточка при наведении

## 🌳 ВИЗУАЛИЗАЦИЯ TREE-HEALTH

- CSS-дерево на странице учения — 7 цветных блоков
- Лес учений — отдельная страница со всеми 254 деревьями, фильтр по цвету
- Радар-граф для стран — 7 осей (из чекера стран)

## 🔍 ПОИСК И ФИЛЬТРАЦИЯ

- Полнотекстовый поиск через `golem.db`
- Подсветка найденных слов в тексте
- Поиск по стихам: «Берешит 1:1»

## 📖 ЧТЕНИЕ И КОМФОРТ

- Режим чтения — скрыть сайдбар и всё лишнее
- Прогресс чтения — полоска вверху
- Сноски при наведении на стих — всплывашка с полным текстом
- Ночной режим по расписанию (после заката)
- Размер шрифта: мелкий / средний / крупный
