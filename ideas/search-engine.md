## 🔥 ЧТО ЭТО

Поисковый движок — инструмент для быстрого поиска по всем текстам репозитория и ТаНаХа.

**Зачем нужен:**
- найти все вхождения слова или корня
- быстро найти нужный термин
- искать по всем исследованиям сразу
- найти стихи по ключевым словам

---

## 🎯 ТИПЫ ПОИСКА

**1. ПОЛНОТЕКСТОВЫЙ ПОИСК**

Ищет по всем md-файлам репозитория.

Пример: «хесед» → все файлы, где есть это слово

**2. ПОИСК ПО ТЕРМИНАМ**

Ищет только в папке `terminology/`.


**3. ПОИСК ПО ИССЛЕДОВАНИЯМ**

Ищет только в папке `researches/`.

Пример: «Вавилон» → все исследования о Вавилоне

**4. ПОИСК ПО ТАНАХУ**

Ищет по стихам ТаНаХа (иврит, транслитерация, перевод).

Пример: «שמע» → все стихи со словом «шма»

**5. ПОИСК ПО КОРНЯМ**

Ищет все слова с указанным корнем.

Пример: «אהב» → аhава, аhув, аhавим

**6. ПОИСК ПО ПОДМЕНАМ**

Ищет религионизмы и показывает ивритский оригинал.

Пример: «Господь» → Яхве

**7. ПОИСК ПО ССЫЛКАМ**

Находит все файлы, которые ссылаются на указанный файл.

Пример: «emet.md» → все файлы, где есть ссылка на emet.md

## 🛠️ ТЕХНИЧЕСКАЯ РЕАЛИЗАЦИЯ

**ИНДЕКСАЦИЯ:**

При первом запуске сканируются все md-файлы и создаётся индекс.

```bash
python3 tools/build-search-index.py
```

**ИНДЕКС (JSON):**
```json
{
  "хесед": {
    "terminology/chesed.md": [10, 15, 42],
    "researches/brit.md": [5, 20],
    "README.md": [30]
  }
}
```

**ПОИСК:**
```bash
python3 tools/search.py "хесед"
python3 tools/search.py --type tanakh "שמע"
python3 tools/search.py --type root "אהב"
python3 tools/search.py --type reverse "Господь"
```

**ФОРМАТ ВЫВОДА:**

```
🔍 РЕЗУЛЬТАТЫ ПОИСКА: "хесед"
================================

📁 terminology/chesed.md
   строка 10: Хесед — преданная любовь

   строка 5: Хесед — основа союза

📁 README.md
   строка 30: термины: хесед, эмет, тшува

================================
Найдено: 3 файла, 4 вхождения
```

## 💡 ПРИМЕРЫ ИСПОЛЬЗОВАНИЯ

**ПОИСК СЛОВА В ТАНАХЕ:**
```bash
python3 tools/search.py --type tanakh "אֱמֶת"
```

Результат:
- Теhилим 119:160 — «Истина Твоя в род и род»
- Берешит 24:48 — «Истиной Яхве»
- Йешаяhу 65:16 — «Яхве истины»

**ПОИСК СТИХА ПО ССЫЛКЕ:**
```bash
python3 tools/search.py --type verse "Берешит 1:1"
```

Результат:
- показывает иврит, транслитерацию, перевод
- ссылки на исследования об этом стихе

**ПОИСК РЕЛИГИОНИЗМА:**
```bash
python3 tools/search.py --type reverse "Бог"
```

Результат:
- замены: «Элоhим» или «Яхве»
- предупреждение: религионизм
- список файлов с этим словом

**ПОИСК ПО ГЛОССАРИЮ:**
```bash
python3 tools/search.py --type glossary "и"
```

Результат:
- все термины, начинающиеся на «и»
- истина, имя, избавление
