# Палео-механика

Не словарь. Не перевод. Чертёжная мастерская.

Каждая буква палео-иврита здесь разбирается как функция, а не как образ-иллюстрация. Буква — не «означает быка» или «означает дом». Буква работает: открывает, соединяет, перемалывает, удерживает, отделяет.

Раздел дополняет [Словарь терминов](../../03-CONTENT/MANIFEST.md) манифеста — там буквы даны как список соответствий, здесь — как разобранный механизм.

## Структура файла буквы

- **Образ** — конкретный предмет из быта кочевника
- **Функция** — что буква делает, а не что она «значит»
- **Палео-написание** — символ
- **Как работает** — механика в 2–4 пунктах
- **Пример в слове** — разбор корня с этой буквой
- **Сравнение с греческой подменой** — если есть аналогичное искажение в переводческой традиции
