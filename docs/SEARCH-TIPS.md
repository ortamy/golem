## 🎯 ТРИ СПОСОБА ПОИСКА

### Веб-интерфейс
- **Скорость:** мгновенно
- **Где ищет:** заголовки, темы
- **Когда использовать:** найти файл по названию
- **Плюсы:** мгновенно, фильтры по категориям
- **Минусы:** только заголовки и темы, не ищет внутри текста

### search.py
- **Скорость:** быстро
- **Где ищет:** содержимое всех `.md` файлов
- **Когда использовать:** найти где упоминается слово или фраза
- **Плюсы:** ищет внутри текста, показывает сниппеты с контекстом
- **Минусы:** медленнее веба, только `.md` файлы

### grep
- **Скорость:** мгновенно
- **Где ищет:** содержимое всех файлов
- **Когда использовать:** найти точное совпадение, посчитать вхождения
- **Плюсы:** мгновенно, любые файлы, поддержка regex
- **Минусы:** только точное совпадение, нет сниппетов

## 🌐 ВЕБ-ИНТЕРФЕЙС

Поиск по заголовкам и темам файлов. Не ищет внутри текста.

```bash
http://localhost:8080
```

Ввести слово в поле «Поиск» в сайдбаре.

**Ограничения:**
- Только заголовки и темы
- Не понимает иврит

## 🔍 SEARCH.PY

Поиск по содержимому всех `.md` файлов.

```bash
# Базовый поиск
python tools/utils/search.py "хесед"

# Ограничить количество результатов
python tools/utils/search.py "преданная любовь" --limit 5

# Показать контекст вокруг каждого совпадения
python tools/utils/search.py "банк" --verbose
```

**Опции:**
- `--limit N` — показать только N результатов
- `--verbose` — показать все сниппеты с контекстом

**Особенности:**
- Игнорирует регистр
- Понимает иврит
- Ищет только в `.md` файлах

## ⚡ GREP

Мгновенный поиск по всем файлам проекта.

```bash
# Найти слово во всех .md файлах
grep -r "кастрация смысла" --include="*.md"

# Только имена файлов (без содержимого)
grep -rl "Яхве" --include="*.md"

# Показать номера строк
grep -rn "Элоhим" --include="*.md"

# Искать только в конкретной папке
grep -r "подмена" researches/tanakh/

# Искать в .py файлах
grep -r "SCAN_DIRS" tools/ --include="*.py"

# Посчитать количество вхождений
grep -rc "Яхве" --include="*.md" | sort -t: -k2 -rn | head -20

# Искать с учётом регистра
grep -ri "яхве" --include="*.md"
```

**Полезные флаги:**
- `-r` — рекурсивно по папкам
- `-l` — только имена файлов
- `-n` — номера строк
- `-i` — игнорировать регистр
- `-c` — посчитать вхождения
- `--include="*.md"` — фильтр по расширению

## 💡 СОВЕТЫ

### Понимание особенностей
- **search.py** игнорирует регистр, **grep** — нет (используй `-i`)
- **search.py** и **grep** понимают иврит, **веб** — нет
- **grep** быстрее всех, **веб** мгновенный для заголовков

### Продвинутый grep
```bash
# Искать несколько слов (ИЛИ)
grep -r "подмена\|искажение" --include="*.md"

# Искать файлы, содержащие оба слова (И)
grep -rl "Яхве" --include="*.md" | xargs grep -l "Машиах"

# Исключить папку из поиска
grep -r "поиск" --exclude-dir=tools/cache --include="*.md"
```

### Регулярные выражения
```bash
# Найти «подмена» + любое слово + «смысла»
grep -r "подмена.*смысла" --include="*.md"

# Найти строки с двумя и более цифрами подряд
grep -rn "[0-9]\{2,\}" --include="*.md"
```

### Поиск по метаданным
```bash
# Найти все файлы с определённой темой
grep -r "\*\*Тема:\*\*.*Шаббат" --include="*.md" -l

# Найти все файлы в статусе «Черновик»
grep -r "\*\*Статус:\*\*.*Черновик" --include="*.md" -l
```
