## 🌐 ВЕБ-ИНТЕРФЕЙС

Поиск по заголовкам и темам файлов. Не ищет внутри текста.

```
http://localhost:8080
```

Ввести слово в поле «Поиск» в сайдбаре.

**Плюсы:** мгновенно, фильтры по категориям
**Минусы:** только заголовки и темы

---

## 🔍 SEARCH.PY

Поиск по содержимому всех `.md` файлов.

```bash
python tools/utils/search.py "хесед"
python tools/utils/search.py "преданная любовь" --limit 5
python tools/utils/search.py "банк" --verbose
```

**Плюсы:** ищет внутри текста, показывает сниппеты
**Минусы:** медленнее веба, только .md файлы

**Опции:**
- `--limit N` — показать только N результатов
- `--verbose` — показать все сниппеты для каждого файла

## ⚡ GREP

Мгновенный поиск по всем файлам проекта.

```bash
# Найти слово во всех .md файлах
grep -r "кастрация смысла" --include="*.md"

# Показать имена файлов
grep -rl "Яхве" --include="*.md"

# Показать номера строк
grep -rn "Элоhим" --include="*.md"

# Искать только в папке
grep -r "подмена" researches/tanakh/

# Искать в .py файлах
grep -r "SCAN_DIRS" tools/ --include="*.py"
```

**Плюсы:** мгновенно, любые файлы
**Минусы:** только точное совпадение, нет сниппетов

## 🎯 КОГДА ЧТО ИСПОЛЬЗОВАТЬ

- «Найти исследование про шаббат» → **Веб**
- «Где упоминается слово хесед?» → **search.py**
- «В каком файле строка SCAN_DIRS?» → **grep**
- «Сколько раз встречается слово Яхве?» → **grep -c**
- «Показать контекст вокруг слова» → **search.py --verbose**

---

## 💡 СОВЕТЫ

- **Иврит:** search.py и grep понимают иврит. Веб — нет.
- **Регистр:** search.py игнорирует регистр. grep — нет (используй `-i`)
- **Скорость:** grep быстрее всех. Веб мгновенный для заголовков.
- **Регулярки:** grep поддерживает regex: `grep -r "подмена.*смысла"`
