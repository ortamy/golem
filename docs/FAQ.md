## 🔥 ОБЩИЕ ВОПРОСЫ

### Что такое «Голем»?
Проект восстановления истины ТаНаХа. Мы разоблачаем подмены в переводах и возвращаем словам их ивритские корни. Не религия. Не церковь. Исследовательский проект.

### Почему «Яхве», а не «Господь» или «Бог»?
Потому что יהוה — это личное имя. «Господь» и «Бог» — титулы, которыми заменили Имя. Это как называть человека «господин» вместо имени.

### Почему вы против таблиц?
Таблицы — римский формат контроля. Истина не укладывается в колонки. Списки — свобода. Всё, что можно сказать таблицей, можно сказать списком.

## 🔧 ТЕХНИЧЕСКИЕ ВОПРОСЫ

### Скрипт упал с ошибкой. Что делать?
```bash
# 1. Очистить кэш
python tools/checkers/clear-cache.py

# 2. Перестроить кэш словарей
python tools/checkers/check-religionisms.py --rebuild

# 3. Если не помогло — проверить логи
cat golem.log
```

### Почему все проверки показывают 0 файлов?
Скрипты не находят папки для сканирования. Проверь, что запускаешь из корня репозитория (`golem-main/`), а не из `tools/`.

### Как добавить новое запрещённое слово?
Добавь его в соответствующий файл в `instructions/tahor/`. Потом:
```bash
python tools/checkers/clear-cache.py
python tools/checkers/check-religionisms.py --rebuild --fix
```

### Как откатить изменения после `--fix`?
```bash
git checkout -- .
```
Или восстановить конкретный файл:
```bash
git checkout -- terminology/yhwh.md
```

### Не устанавливается `windows-curses`. Что делать?
```bash
pip install windows-curses
```
Если ошибка — установи Visual C++ Build Tools или используй WSL.

### Как запустить проект на Linux/Mac?
```bash
pip install rich
cd tools && python golem.py
```
`curses` встроен в Python на Linux/Mac.

## 📝 РАБОТА С ФАЙЛАМИ

### Куда положить новый файл?
Если это термин — в `terminology/`. Если исследование — в `researches/` (подпапку по теме). Если инструкция — в `instructions/`. Если не знаешь — положи в `drafts/` и спроси.

### Как правильно назвать файл?
Только латиница, цифры и дефис. Никаких русских букв, пробелов, подчёркиваний. Пример: `my-research.md`.

Ивритские темы — ивритское имя: `chesed.md`, `brit-hadashah.md`. Светские темы — английское: `history-of-banks.md`, `psychology.md`.

### Как заполнить метаданные?
Запусти `unify-metadata.py --fix` — он добавит недостающие поля. Или создай вручную по шаблону из `instructions/templates/research-template.md`.

### Что делать с пустым файлом?
Запусти `fill-empty-files.py` — он создаст черновик с правильной структурой. Потом заполни содержанием.

## 🔍 ПРОВЕРКИ

### Какие проверки запускать ежедневно?
```bash
python tools/checkers/check-links.py
python tools/checkers/find-orphans.py
```

### Какие проверки запускать перед коммитом?
```bash
python tools/checkers/check-naming.py
python tools/checkers/validate-metadata.py --fix
python tools/checkers/check-religionisms.py --fix
```

### Почему `check-religionisms.py` работает долго?
Первый запуск сканирует все 2000+ файлов. Второй запуск — только изменённые (в 10-50 раз быстрее). Используй `--fix` для ускорения.

### Что делать если скрипт находит «Тору» как религионим?
Это ложное срабатывание. Убедись, что кэш версии 3.1+ (`--rebuild`). Ивритские слова в белом списке.

## 📖 ТЕРМИНОЛОГИЯ

### Что такое «религионизм»?
Система, имитирующая истину. Каждая сфера вне Яхве — алтарь, жертва, жрец. Карьера, наука, спорт — всё может быть религионизмом.

### Что такое «bdikah», «mivdak», «tikun», «factcheck»?
Четыре этапа аудита: bdikah — проверка на религионимы, mivdak — оценка полезности, tikun — задачи по обновлению, factcheck — проверка фактов.

### Почему «эмуна», а не «вера»?
Потому что אמונה — это верность в делах. Не внутреннее чувство. Верность можно проверить. «Веру» — нет.

### Почему «тшува», а не «покаяние»?
Потому что תשובה — это физический разворот. Не чувство вины. Развернулся и пошёл обратно. Это видно.

## 🛠 GOLEM.PY

### Как добавить новый скрипт в меню?
Просто положи его в `tools/checkers/`, `tools/generators/`, `tools/reports/` или `tools/automation/`. Меню обновится автоматически.

### Как сменить язык меню?
При запуске `golem.py` выбери язык (1 — Русский, 2 — English). Или измени `golem-config.json` в `tools/cache/`.

### Меню зависло. Что делать?
Нажми `q` или `Esc`. Если не помогает — `Ctrl+C` в терминале.
