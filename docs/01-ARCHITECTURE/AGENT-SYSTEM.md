# Система AI-агентов

## Категории агентов

### Исследователь
- Исследователь — разбор корней, стихов, терминов
- Разоблачитель — поиск подмен в переводах
- Сборщик — объединение результатов в отчёт
- Семитолог — прасемитские параллели
- Компаратор — трёхстороннее сравнение текстов
- Переводчик палео-иврита — буква → образ → смысл

### Разработчик
- Фронтенд-разработчик — Vanilla JS, CSS, SVG
- AI-инженер — PyTorch, Flask API

### Контроль
- Критик — проверка соответствия методологии
- Проверяющий — валидация кода и гипотез
- Ревьюер кода — проверка по MANIFEST.md

### Документация
- Редактор — стиль проекта, убрать религионимы
- Технический писатель — Markdown, ссылки на docs

### Оркестрация
- Архитектор потока — порядок вызова агентов
- Связной — связи между исследованиями
