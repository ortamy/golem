## 📜 СЛОВАРЬ

### Связь и общение

- **Подключение** (connect) — установка связи → קָשַׁר (кашар) — связывать. Или: חִבֵּר (хибер) — соединять. Подмена: живая связь заменена на техническое подключение
- **Сеть** (network) — система связей → רֶשֶׁת (решет) — сеть. ТаНаХ: «Нечестивый сетью уловляет людей» (Мишлей 12:12). Подмена: орудие ловли стало идеалом связи
- **Социальная сеть** — social network → קְהִלָּה (кеhила) — собрание. Подмена: живое собрание заменено на цифровую сеть
- **Интерфейс** — способ взаимодействия → פָּנִים (паним) — лицо. Подмена: лицом к лицу заменено на интерфейс
- **Протокол** — правила передачи данных → דֶּרֶךְ (дэрех) — путь. Подмена: путь заменён на технический протокол
- **Трафик** — поток данных → דֶּרֶךְ (дэрех) — путь. Или: מַעֲבָר (маавар) — переход
- **Сигнал** — передаваемый знак → אוֹת (от) — знак. Подмена: знак от Яхве заменён на технический сигнал
- **Пинг** — проверка связи → קָרָא (кара) — звать. Подмена: зов заменён на технический пинг

### Информация и знание

- **Контент** — содержимое → תֹּכֶן (тохен) — содержание. Или: דָּבָר (давар) — слово, дело
- **Данные** — информация → דַּעַת (даат) — знание. Подмена: живое знание заменено на мёртвые данные
- **Алгоритм** — последовательность действий → חֹק (хок) — устав. Или: מִשְׁפָּט (мишпат) — порядок. Подмена: порядок Яхве заменён на алгоритм
- **Поиск** — search → דָּרַשׁ (дараш) — искать, исследовать. Подмена: исследование Торы заменено на поиск в интернете
- **Браузер** — обозреватель → צוֹפֶה (цофе) — наблюдатель, страж. Подмена: страж на стене заменён на обозреватель страниц
- **Скачать** (download) — загрузить → לָקַח (лаках) — брать. Или: קִבֵּל (кибель) — получать
- **Загрузить** (upload) — передать → נָתַן (натан) — дать. Или: שָׁלַח (шалах) — послать
- **Файл** — единица хранения → מְגִלָּה (мегила) — свиток. Подмена: свиток заменён на файл
- **Папка** (folder) — каталог → תֵּבָה (тева) — ящик, ковчег. Или: אֲרוֹן (арон) — ящик
- **Архив** — хранилище → אוֹצָר (оцар) — сокровищница. Подмена: сокровищница мудрости заменена на технический архив

### Платформы и системы

- **Платформа** — основа → מָקוֹם (маком) — место. Или: יְסוֹד (йесод) — основание. Подмена: основание на камне заменено на цифровую платформу
- **Система** — упорядоченное целое → סֵדֶר (седер) — порядок. Подмена: порядок Яхве заменён на техническую систему
- **Приложение** (app) — программа → כְּלִי (кли) — инструмент, сосуд. Подмена: инструмент заменён на приложение
- **Сервис** — услуга → עֲבוֹדָה (авода) — служение, работа. Подмена: служение заменено на сервис
- **Аккаунт** — учётная запись → שֵׁם (шем) — имя. Подмена: имя заменено на безличный аккаунт

- **Подписка** — регулярный доступ → בְּרִית (брит) — союз. Подмена: союз заменён на платную подписку
- **Облако** (cloud) — удалённое хранилище → שָׁמַיִם (шамаим) — небеса. Подмена: небеса Яхве заменены на цифровое облако

### Действия и процессы

- **Логин** — вход в систему → בּוֹא (бо) — входить. Или: עָבַר (авар) — переходить
- **Аутентификация** — подтверждение личности → אֱמֶת (эмет) — истина, подтверждение. Подмена: подтверждение истины заменено на техническую проверку
