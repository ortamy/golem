## СТАТЬИ СЛОВАРЯ

### АДАМА (ЗЕМЛЯ)

- **Ивритский оригинал:** אֲדָמָה (Адама)
- **Палео-образы:** א (сила) + ד (дверь) + מ (вода/жизнь) + ה (откровение)
- **Визуальная сборка:** сила, открывающая дверь жизни через окно
- **Функция:** материал, из которого слеплена форма
- **Корень:** א.ד.מ — быть красным, глинистым
- **Живой образ:** глина, из которой гончар лепит сосуд
- **Тип искажения:** абстракция + сужение смысла
- **Механизм подмены:** Подмена палео-механики на греческую физику (Механизм 7)
- **Метод разоблачения:** Палео-механический разбор (Метод 1) + Конкретизация (Метод 4)
- **Карта утрат:** адама (глина) → γῆ (стихия) → «земля» (планета) → «почва» (абстрактный объект)
- **Разбор:** Адама — это не «планета Земля» в космическом смысле. Это глина. Красная глина. Это материал, из которого сделан человек. Когда Яхве говорит «Адама», он говорит о конкретной земле под ногами, о материале, из которого ты слеплен. Греки превратили это в γῆ — «земля как планета», абстрактный объект в космосе.
- **Стихи:** Берешит 2:7 («прах от адама»), Берешит 3:19 («возвратишься в адама»)
- **Пример восстановления:** «Адам был взят из глины» вместо «Адам был взят из земли».

### АНГЕЛ

- **Ивритский оригинал:** מַלְאָךְ (малъах)
- **Палео-образы:** מ (вода) + ל (посох) + א (сила) + כ (ладонь)
- **Визуальная сборка:** поток, направляемый силой через ладонь
- **Функция:** вестник (тот, кто передаёт поручение)
- **Корень:** ל.א.כ — посылать, передавать
- **Живой образ:** гонец, который бежит с письмом
- **Тип искажения:** подмена категории (функция → существо)
- **Механизм подмены:** Подмена живого отношения на мёртвую структуру (Механизм 4)
- **Метод разоблачения:** Функциональное опознание + Морфологическое вскрытие
- **Карта утрат:** малъах (вестник) → ἄγγελος (посланник) → «ангел» (существо) → «крылатое создание» (мифология)
- **Разбор:** малъах — не вид существа. Это функция «вестник». Исполнять может человек (Хагай 1:13 — пророк назван малъах), ветер (Теhилим 104:4 — ветры названы малъахав), огонь. Система превратила функцию в расу крылатых созданий. В иврите нет слова «ангел» как существа. Есть вестник как поручение.
- **Стихи:** Хагай 1:13 («малъах Яхве»), Теhилим 104:4 («ветры — малъахав»)
- **Пример восстановления:** «И сказал Хагай, вестник Яхве» вместо «Аггей, ангел Господень».

### БЛАГОДАТЬ

- **Ивритский оригинал:** חֵן (хэн)
- **Палео-образы:** ח (ограда/отделение) + נ (рыба/жизнь)
- **Визуальная сборка:** ограда, отделяющая жизнь
- **Функция:** расположение, благосклонность
- **Корень:** ח.נ.נ — быть благосклонным, склоняться
- **Живой образ:** человек, который склоняется к другому
- **Тип искажения:** психологизация + абстракция
- **Механизм подмены:** Подмена многомерной истины плоским суррогатом (Механизм 3)
- **Метод разоблачения:** Образное картирование + Этимологическая хирургия
- **Карта утрат:** хэн (склонение) → χάρις (дар) → «благодать» (энергия) → «милость» (чувство)
- **Разбор:** хэн — не «благодать» как безличная сила. Это склонение. Яхве склоняется к человеку. Благосклонность. Расположение. Конкретное отношение, а не абстрактная субстанция. Система сделала из этого «энергию спасения», которую можно «получить».
- **Стихи:** Берешит 6:8 («Ноах нашёл хэн в глазах Яхве»)
- **Пример восстановления:** «Ноах нашёл расположение в глазах Яхве» вместо «обрёл благодать пред очами Господа».

### БЛАГОСЛОВЕНИЕ

- **Ивритский оригинал:** בְּרָכָה (браха)
- **Палео-образы:** ב (дом) + ר (голова) + כ (ладонь) + ה (откровение)
- **Визуальная сборка:** дом, который голова направляет через ладонь к откровению
- **Функция:** наделение силой
- **Корень:** ב.ר.כ — преклонять колено, наделять силой
- **Живой образ:** отец, который передаёт власть сыну
- **Тип искажения:** психологизация + сдвиг от действия к эмоции
- **Механизм подмены:** Подмена многомерной истины плоским суррогатом (Механизм 3)
- **Метод разоблачения:** Этимологическая хирургия + Морфологическое вскрытие
- **Карта утрат:** браха (наделение силой) → εὐλογία (хорошее слово) → «благословение» (пожелание) → «благодарность» (чувство)
- **Разбор:** браха — не «хорошее пожелание». Это наделение силой. Колено преклоняется — передаётся сила. Конкретное действие. Яхве благословляет — даёт способность. Отец благословляет сына — передаёт власть и наследие. Система сделала из этого «доброе слово».
- **Стихи:** Берешит 27:27 («Иаков получил браха от отца»)
- **Пример восстановления:** «И наделил их силой Всесильный» вместо «и благословил их Бог».

### БОГ / ГОСПОДЬ

- **Ивритский оригинал:** אֱלֹהִים (Элоhим) / יְהוָה (Яхве)
- **Палео-образы (Элоhим):** א (сила) + ל (посох/направление) + ה (дыхание/откровение) + י (рука/действие) + מ (вода/жизнь)
- **Визуальная сборка (Элоhим):** сила, направляющая дыхание через руку в поток
- **Палео-образы (Яхве):** י (рука) + ה (дыхание) + ו (крюк/связь) + ה (дыхание)
- **Визуальная сборка (Яхве):** рука, связывающая дыхание с дыханием
- **Функция (Элоhим):** полнота сил
- **Функция (Яхве):** алгоритм, связывающий действие и проявление
- **Корни:** א.ל.ה — быть сильным; ה.י.ה — быть, существовать, становиться
- **Живой образ (Элоhим):** несколько сил, работающих вместе
- **Живой образ (Яхве):** конструкция, которая соединяет действие и проявление
- **Тип искажения:** подмена имени на титул + сужение смысла
- **Механизм подмены:** Религиозный синкретизм (Механизм 6) + Подмена палео-механики на греческую физику (Механизм 7)
- **Метод разоблачения:** Разрыв цепочки перевода + Морфологическое вскрытие
- **Карта утрат:** Элоhим/Яхве → θεός (божество) → Deus (римский бог) → «Бог»/«Господь» (титул)
- **Разбор:** Элоhим — форма множественного числа. Множественное величия. Полнота сил. В палео-иврите: бык (сила) + посох (власть) + дыхание (откровение) + рука (действие) + вода (жизнь). Яхве — личное имя, которое означает «Тот, Кто был, есть и будет» (алгоритм бытия). Система заменила имя и силу на греческий θεός (божество) и латинский Dominus (господин). Слово «Бог» — славянское, языческий корень, означает «наделяющий долей», связано с пантеоном.
- **Стихи:** Берешит 1:1 («Элоhим»), Шмот 3:14 («Яхве»)
- **Пример восстановления:** оставить «Элоhим» и «Яхве» без перевода. В крайнем случае — «Всесильный» и «Сущий».

### ВЕРА

- **Ивритский оригинал:** אֱמוּנָה (эмуна)
- **Палео-образы:** א (сила) + מ (вода/жизнь) + ו (крюк/связь) + נ (рыба/жизнь) + ה (откровение)
- **Визуальная сборка:** сила, текущая жизнью, соединённая с живым, в откровении
- **Функция:** твёрдость, надёжность, верность в действии
- **Корень:** א.מ.נ — быть твёрдым, надёжным, верным
- **Живой образ:** человек, который стоит твёрдо и не шатается
- **Тип искажения:** психологизация + сдвиг от действия к эмоции
- **Механизм подмены:** Подмена многомерной истины плоским суррогатом (Механизм 3)
- **Метод разоблачения:** Этимологическая хирургия + Семантическая реконструкция
- **Карта утрат:** эмуна (твёрдость) → πίστις (доверие) → «вера» (мнение) → «убеждение» (чувство)
- **Разбор:** эмуна — не «вера» как мнение. Это верность. Твёрдость. Надёжность. От корня амен — быть прочным. Эмуна — это когда ты стоишь твёрдо, держишь слово, не шатаешься. Система заменила действие на чувство: «верить» можно лёжа на диване. «Быть верным» — нельзя.
- **Стихи:** Берешит 15:6 («он был твёрд в Яхве»)
- **Пример восстановления:** «Праведный своей верностью жив будет» вместо «верою жив будет».

### ВЕЧНОСТЬ

- **Ивритский оригинал:** עוֹלָם (олам)
- **Палео-образы:** ע (глаз/видение) + ו (крюк/связь) + ל (посох/направление) + מ (вода/жизнь)
- **Визуальная сборка:** глаз, смотрящий на крюк, который направляет поток к скрытому
- **Функция:** скрытый горизонт
- **Корень:** ע.ל.מ — скрывать, быть тайным
- **Живой образ:** горизонт, за который не видно
- **Тип искажения:** абстракция + вавилонизация
- **Механизм подмены:** Подмена многомерной истины плоским суррогатом (Механизм 3) + Религиозный синкретизм (Механизм 6)
- **Метод разоблачения:** Сравнительная семитология + Образное картирование
- **Карта утрат:** олам (скрытый горизонт) → αἰών (век/эон) → «вечность» (бесконечное время) → «вечное» (абстракция)
- **Разбор:** олам — не «вечность» в греческом смысле бесконечного времени. Это скрытый горизонт. То, что за пределами видимого. В аккадском: ullu — далёкое время. В арабском: عالم (алям) — мир. Олам — это не линия времени. Это горизонт, за который не видно. Яхве — Эль Олам. Не «вечный Бог». Всесильный скрытого горизонта.
- **Стихи:** Берешит 21:33 («Яхве — Эль Олам»)
- **Пример восстановления:** «Всесильный скрытого горизонта» вместо «Бог вечный».

### ВОДА (МАИМ)

- **Ивритский оригинал:** מַיִם (Маим)
- **Палео-образы:** מ (поток) + י (рука/действие) + מ (поток)
- **Визуальная сборка:** поток, который течёт через действие руки в поток
- **Функция:** текучая жизнь
- **Корень:** מ.י.מ — течь, струиться
- **Живой образ:** вода, которая течёт в реке
- **Тип искажения:** абстракция + сужение смысла
- **Механизм подмены:** Подмена палео-механики на греческую физику (Механизм 7)
- **Метод разоблачения:** Палео-механический разбор (Метод 1) + Конкретизация (Метод 4)
- **Карта утрат:** маим (поток) → ὕδωρ (стихия) → «вода» (H₂O) → «жидкость» (химия)
- **Разбор:** Маим — это не H₂O, не химическое соединение. Это поток. Движение. Жизнь, текущая сквозь. В палео-иврите: поток (мем) + рука (йод) + поток (мем) — «движение, приводимое в действие рукой и текущее дальше». Греки превратили это в ὕδωρ — «вода как стихия». Наука превратила это в H₂O. Потерян поток, потеряна жизнь.
- **Стихи:** Берешит 1:2 («над маим»), Берешит 1:6 («посреди маим»)
- **Пример восстановления:** «Дух Яхве витал над потоком» вместо «над водами».

### ГРЕХ

- **Ивритский оригинал:** חֵטְא (хэт)
- **Палео-образы:** ח (ограда/отделение) + ט (змея/оборачивание) + א (сила/бык)
- **Визуальная сборка:** ограда, которая оборачивает силу
- **Функция:** промах, непопадание в цель
- **Корень:** ח.ט.א — промахиваться, не попадать в цель
- **Живой образ:** стрела, которая пролетела мимо цели
- **Тип искажения:** юридизация + психологизация
- **Механизм подмены:** Затуманивание ясного различия (Механизм 1) + Подмена многомерной истины плоским суррогатом (Механизм 3)
- **Метод разоблачения:** Этимологическая хирургия + Образное картирование
- **Карта утрат:** хэт (промах) → ἁμαρτία (ошибка) → peccatum (преступление) → «грех» (моральное нарушение)
- **Разбор:** хэт — не «грех» как преступление закона. Это промах. Как стрела мимо цели. Ты целился, но не попал. Не злоба. Слабость. Ошибка. Система превратила промах в юридическое преступление, за которое положено наказание. А промах требует исправления, а не кары.
- **Стихи:** Вайикра 4:2 («если душа согрешит по ошибке»), Берешит 4:7 («у дверей грех лежит»)
- **Пример восстановления:** «промах» или «непопадание в цель» вместо «грех».

### ДАВАР (СЛОВО)

- **Ивритский оригинал:** דָּבָר (Давар)
- **Палео-образы:** ד (дверь/вход) + ב (дом/вместилище) + ר (голова/вершина)
- **Визуальная сборка:** дверь, открывающая дом, чтобы выпустить содержимое через голову
- **Функция:** действие через открытие двери
- **Корень:** ד.ב.ר — говорить, открывать, выпускать
- **Живой образ:** человек, который открывает дверь, чтобы выпустить то, что внутри
- **Тип искажения:** подмена категории + сужение смысла
- **Механизм подмены:** Подмена палео-механики на греческую физику (Механизм 7)
- **Метод разоблачения:** Палео-механический разбор (Метод 1) + Возвращение Давара (Метод 34)
- **Карта утрат:** давар (открытие двери) → λόγος (мысль/речь) → «слово» (звук) → «речь» (произнесение)
- **Разбор:** Давар — это не звук. Не «слово» как произнесённая речь. Это открытие двери (Далет), чтобы выпустить содержимое дома (Бет) через голову (Реш). Греческий λόγος — «мысль, выраженная в речи». Ивритский Давар — это действие, а не мысль. Система заменила действие на понятие.
- **Стихи:** Берешит 1:3 («ва-йомэр — и сказал»), Иер. 1:4 («давар Яхве»)
- **Пример восстановления:** «Давар Яхве» — «действие Яхве» вместо «Слово Божие».

### ДУША

- **Ивритский оригинал:** נֶפֶשׁ (нэфеш)
- **Палео-образы:** נ (рыба/жизнь) + פ (рот/дыхание) + ש (зуб/огонь)

### ШЕКЕР (שקר)

- **Ивритский оригинал:** שֶׁקֶר (Шекер)
- **Палео-образы:** ש (зуб/огонь/разрушение) + ק (игла/отделённость) + ר (голова/направление)
- **Визуальная сборка:** направление, пронзённое и сдвинутое
- **Функция:** искажение, перенаправление
- **Корень:** ש.ק.ר — быть ложным, искажённым
- **Живой образ:** рыба, которую выловили крюком из естественного потока и бросили в искусственный пруд. Направление изменено.
- **Тип искажения:** абстракция + сужение смысла
- **Механизм подмены:** Затуманивание ясного различия (Механизм 1) + Религиозный синкретизм (Механизм 6)
- **Метод разоблачения:** Этимологическая хирургия + Метод живого образа (Метод 36)
- **Карта утрат:** шекер (искажение) → ψεῦδος (ложь) → «ложь» (обман) → «фальшь» (мораль)
- **Разбор:** Шекер — это не «ложь» в моральном смысле. Это **перенаправление конструкции** из естественного русла в искусственный канал. Конструкция сдвигается, направление меняется. Это когда что-то сдвигается с правильного русла (Реш) через внешнее вмешательство (Шин + Коф).
- **Противоположность:** Эмет (אמת) — конструкция в правильном русле.
- **Пример восстановления:** «Это не шекер» — «Это не искажение» вместо «Это не ложь».
