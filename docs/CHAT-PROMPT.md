## 📜 О ПРОЕКТЕ «ГОЛЕМ»

Мы строим проект «Голем» — сайт, web app, инструменты для восстановления истины ТаНаХа.

### Что уже есть:
- Сайт на GitHub Pages: `https://ortamy.github.io/golem/`
- Сотни md-файлов с исследованиями в `content/researches/`, `content/terminology/`, `content/tanakh/`, `content/exposed/`
- Локальный сервер `products/website/server.js` (Node.js, статика, порт 8080)
- Страница исследований с поиском и фильтрами
- Страницы: Методы разоблачения, Словари, Методология
- Подстрочник ТаНаХа: `products/website/ru/interlinear/index.html`
- Скрипты в `tools/`: генератор подстрочника, конвертер палео-иврита, чекеры md-файлов
- Дизайн-система: древняя библиотека, скрипторий, пергамент, золото, коричневый
- Бургер-меню на всех страницах
- Три языка: RU, EN, HE
- UI UX Pro Max установлен для Cline

### Что строим сейчас:
- Подстрочник ТаНаХа с палео-ивритом, масоретским текстом, кумранскими фрагментами, LXX, транслитерацией и переводом
- Интеграция с Sefaria API для получения масоретского текста
- Чекер религионимов в переводе
- Навигация по книгам и главам
- Аудио-озвучка стихов
