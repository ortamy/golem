## 🔥 ЧТО ЭТО

Веб-интерфейс проекта «Голем» — адаптивное одностраничное приложение для чтения исследований и терминов. Работает локально через Node.js-сервер и на GitHub Pages через статический `files.json`.

---

## 🏛 АРХИТЕКТУРА

```
web/
├── index.html          # основной HTML (PC + Mobile)
├── app.js              # логика (v9.1)
├── style.css           # стили (адаптивные)
├── server.js           # Node.js сервер для локальной разработки
├── files.json          # сгенерированный индекс (для GitHub Pages)
├── favicon.svg         # иконка
├── package.json        # зависимости Node.js
└── export/
    ├── dashboard.html  # сгенерированный дашборд
    └── golem-book.html # сгенерированная книга
```

**Два режима работы:**
- Локальный: `server.js` сканирует папки и отдаёт `/api/files` и `/api/file`
- Продакшн: GitHub Pages загружает `files.json` и запрашивает `.md` файлы напрямую

## 📋 ФУНКЦИОНАЛ

### Навигация
- Сайдбар со списком файлов, сгруппированных по категориям
- Поиск по заголовкам, темам и путям
- Фильтр по категориям (Терминология, Исследования)
- Подкатегории для папки «Исследования»
- Оглавление (TOC) — авто-генерация из H2/H3 текущего файла
- Кнопка «Случайный файл»
- Хлебные крошки (путь к файлу)

### Чтение
- Парсер Markdown → HTML (заголовки, списки, цитаты, код, жирный, курсив)
- Блок метаданных автоматически скрывается
- Связанные файлы — кликабельные ссылки внизу страницы
- Копирование цитат по клику на blockquote

### Избранное и история
- Закладки: добавление/удаление через ★, сохранение в localStorage
- История: последние 15 открытых файлов
- Синхронизация между вкладками через localStorage

### Тема
- Светлая и тёмная тема
- Переключение по клику на ◐
- Сохранение выбора в localStorage

### Мобильная версия (≤768px)
- Бургер-меню с навигацией
- Поиск и фильтры на весь экран
- Страница файла — полноэкранный режим с кнопкой «Назад»
- Оптимизированные размеры шрифтов и отступов
- Все функции ПК-версии доступны

### Горячие клавиши
- `/` — фокус на поиск
- `b` — добавить/удалить закладку
- `r` — случайный файл
- `Esc` — закрыть файл (мобильный) или снять фокус с поиска

### Дополнительно
- Анимация загрузки (спиннер)
- Плавные переходы между файлами (fade-in)
- Кнопка «Наверх» при скролле
- Уведомление при копировании ссылки
- Печать текущего файла

## 🚀 ЗАПУСК

### Локально

```bash
cd web
node server.js
```

Открыть: `http://localhost:8080`

**API:**
- `GET /api/files` — список всех файлов с метаданными
- `GET /api/file?path=terminology/yhwh.md` — содержимое файла

### GitHub Pages

```bash
python tools/generators/generate-files-json.py
git add web/files.json && git commit -m "Обновление files.json"
git push
```

GitHub Actions автоматически деплоит на `https://<user>.github.io/golem/`

## 📊 ГЕНЕРАЦИЯ files.json

`files.json` — индекс всех `.md` файлов для продакшн-режима.

**Структура записи:**
```json
{
  "path": "terminology/chesed.md",
  "title": "חֶסֶד (ХЕСЕД) — ПРЕДАННАЯ ЛЮБОВЬ",
  "topic": "Библейское понятие верной, союзной любви",
  "category": "Терминология",
  "subcategory": "",
  "related": ["terminology/ahava.md", "terminology/brit.md"]
}
```

**Генерация:**
```bash
python tools/generators/generate-files-json.py
```

## 🔧 СТРУКТУРА server.js

- Сканирует `terminology/` и `researches/` рекурсивно
- Извлекает заголовок H1, тему из метаданных, связанные файлы
- Игнорирует `README.md`, `STRUCTURE.md`, `GLOSSARY.md`, `CHANGELOG.md`
- Определяет подкатегорию для вложенных папок
- Чистит эмодзи из заголовков
- Отдаёт статические файлы из `web/` и `.md` файлы из корня
