# assets/

Место для визуальных референсов дизайн-системы: скриншоты компонентов, примеры палитры в интерфейсе.

Пока пусто — скриншоты требуют реального рендера в браузере (светлая/тёмная тема, ключевые модули: дашборд, корневой словарь, модалка, мобильный вид). Добавлять по мере необходимости, не как формальность.

Предполагаемое содержимое:
- `palette-light.png`, `palette-dark.png` — свотчи цветов из раздела 1 `DESIGN-SYSTEM.md`
- `components-*.png` — карточки, кнопки, алерты, модалка в контексте страницы
- `mobile-*.png` — бургер-меню, мобильный сайдбар, адаптивные карточки
