## 🔥 ЧТО ТАКОЕ ИИ-АГЕНТ

Автономный помощник для управления проектом «Голем». В отличие от нейросети которая только отвечает на вопросы, агент выполняет действия: запускает чекеры, генерирует отчёты, работает с git.

Понимает задачи на русском языке. Сам решает какие инструменты использовать и в каком порядке.

## 🏛 АРХИТЕКТУРА

```
agent/
├── agent.py      # основной агент (логика, выполнение)
├── tools.py      # описание доступных инструментов
├── memory.py     # память агента (история, контекст)
├── config.yml    # конфигурация
└── memory.json   # файл памяти (создаётся автоматически)
```

## 🚀 ЗАПУСК

```bash
# Интерактивный режим
python agent/agent.py

# Авто-режим (одна задача)
python agent/agent.py --auto "проверь всё и исправь"
```

## 🧠 РЕЖИМЫ РАБОТЫ

### Нейросетевое планирование
Если запущен сервер `ed-neural/inference/server.py`, агент отправляет задачу нейросети. Нейросеть возвращает JSON с планом: какие инструменты запустить, с какими флагами, и почему.

### Планирование по правилам
Если нейросеть недоступна, агент использует встроенные правила по ключевым словам. «проверить религионизмы» → `check-religionisms.py --fix`.

## 📋 ПРИМЕРЫ ЗАДАЧ

Простые:
- «проверь религионизмы и исправь»
- «найди дубликаты»
- «поиск хесед»

Сложные (требуют нейросети):
- «проверь всё и исправь ошибки»
- «найди дубликаты, проверь ссылки и обнови веб-интерфейс»
- «сгенерируй книгу и дашборд, сделай коммит»

## 🧰 ДОСТУПНЫЕ ИНСТРУМЕНТЫ

### Проверки (checkers)
- `check-religionisms` — поиск и исправление религионимов
- `check-links` — проверка битых ссылок
- `check-empty-files` — поиск пустых файлов
- `check-naming` — проверка имён файлов
- `check-code-quality` — проверка качества Python-кода
- `check-exposure` — проверка по exposure-критериям
- `check-duplicates` — поиск дубликатов
- `check-sizes` — анализ размеров файлов
- `check-tanakh-references` — проверка ссылок на ТаНаХ

### Генераторы (generators)
- `generate-book` — HTML-книга из всех исследований
- `generate-files-json` — files.json для веб-интерфейса
- `generate-dashboard` — интерактивный дашборд
- `generate-exposure-suggestions` — поиск новых методов
- `generate-knowledge-cache` — кэш знаний для нейросети

### Утилиты (utils)
- `search` — поиск по всем файлам репозитория
- `rename-script` — переименование скриптов
- `code-injector` — программное редактирование кода

### Git
- `git-status` — статус репозитория
- `git-commit` — создание коммита
