## 🔥 КАК ПОМОЧЬ

Проект «Голем» открыт для участия. Ты можешь помочь даже без навыков программирования.

### Что можно делать

- Писать исследования (`content/researches/`)
- Разбирать термины (`content/terminology/`)
- Находить и исправлять религионизмы
- Создавать идеи (`ideas/`)
- Улучшать документацию (`guides/`, `docs/`)
- Тестировать скрипты и находить ошибки
- Распространять проект

---

## 📋 ПРОЦЕСС

### Для исследований

1. Найди тему — проверь нет ли дубликата: `python tools/utils/search.py "тема"`
2. Создай черновик в `drafts/`
3. Напиши по шаблону `instructions/templates/TEMPLATE-RESEARCH.md`
4. Пройди аудит: проверь религионимы, факты, стилистику
5. Отправь через pull request
6. После проверки файл перенесут в `content/`

### Для идей

1. Создай файл в `ideas/`
2. Опиши идею: что, зачем, как
3. Отправь через pull request

### Для исправлений

1. Найди ошибку
2. Исправь
3. Отправь через pull request
