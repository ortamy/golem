## 🔥 ОБЩИЕ ПРАВИЛА

Все иконки сайта хранятся в папке `web/icons/`. Поддерживаются форматы SVG и PNG. SVG предпочтительнее — масштабируются без потери качества и меняют цвет через CSS.

---

## 📂 ФОРМАТЫ

### SVG (рекомендуется)
- Масштабируется на любой размер без потери качества
- Меняет цвет через `fill="currentColor"` — одна иконка для светлой и тёмной темы
- Весит 1-5 КБ
- Редактируется в любом текстовом редакторе

### PNG
- Фиксированный размер — для каждого нужен отдельный файл
- Не меняет цвет — нужны две версии для светлой и тёмной темы
- Весит больше SVG
- Подходит только если SVG недоступен

---

## 📏 РАЗМЕРЫ

Три стандартных размера. Каждый в своей папке:

```

web/icons/
├── 16/    # меню, кнопки, списки
├── 24/    # заголовки H2-H4, карточки
└── 32/    # заголовок H1, логотип

```

### 16x16 px
Где использовать: кнопки действий, иконки в списке файлов, значки поиска, закладок, меню, темы.

### 24x24 px
Где использовать: заголовки разделов H2-H4, иконки категорий в сайдбаре, иконки связанных файлов.

### 32x32 px
Где использовать: заголовок страницы H1, логотип, favicon.

---

## 📋 ОТСТУПЫ

Иконка должна иметь внутренний отступ от краёв — **padding 2px для 16x16, 3px для 24x24, 4px для 32x32**.

- Для SVG: сам графический элемент не должен касаться краёв viewBox. Например, в иконке 24x24 основной рисунок занимает 18x18, отступ 3px с каждой стороны
- Для PNG: прозрачное поле 2-4px вокруг рисунка

Это гарантирует что иконки визуально одинакового размера в интерфейсе, даже если рисунок разной формы.

---

## 🎨 СТИЛЬ

Все иконки в едином стиле:

- Линии: толщина 2px для 16x16, 2.5px для 24x24, 3px для 32x32
- Цвет: `#d4a843` (золото) — основной. `#ff4d00` (акцентный) — для важных элементов
- Заливка: только контур (outline). Без сплошной заливки
- Стиль: минимализм, чёткие линии, без теней и градиентов

Иконки должны быть узнаваемы в чёрно-белом варианте. Проверка: переведи в монохром — если иконка не читается, упрости.

## 🔌 ПОДКЛЮЧЕНИЕ SVG

### Способ 1: инлайн в HTML (лучший)

```html
<svg width="24" height="24" viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="2">
    <path d="M..."/>
</svg>
```

Плюсы: цвет меняется через CSS автоматически. Иконка грузится мгновенно.

Способ 2: через <img>

```html
<img src="icons/24/scroll.svg" class="icon-h2" alt="Свиток">
```

Плюсы: простота. Минусы: цвет не меняется через CSS. Для тёмной темы нужен отдельный файл.

Способ 3: через CSS background

```css
.icon-scroll {
    width: 24px; height: 24px;
    background: url('icons/24/scroll.svg') no-repeat center;
}
```

Минусы: цвет не меняется. Для тёмной темы — отдельный класс.

---

🎯 CSS КЛАССЫ ДЛЯ ИКОНОК

Базовые классы в style.css:

```css
.icon-16 { width: 16px; height: 16px; flex-shrink: 0; }
.icon-24 { width: 24px; height: 24px; flex-shrink: 0; }
.icon-32 { width: 32px; height: 32px; flex-shrink: 0; }
```

Иконки-кнопки (кликабельные):

```css
.icon-btn {
    cursor: pointer;
    transition: opacity 0.2s ease, transform 0.2s ease;
}
.icon-btn:hover { opacity: 0.8; transform: scale(1.1); }
```
