### Именование

### Структура

- `var` в новых функциях (использовать `let`/`const`)

## 🎨 CSS

### Заголовок файла

```css
/* web/style.css — стили веб-интерфейса Golem */
```

### Структура

- Группировка по компонентам
- Комментарий перед каждой группой
- Отступы: 2 пробела
- Медиа-запросы в конце файла
- Сброс стилей в начале (`* { margin: 0; padding: 0; box-sizing: border-box; }`)

### Анимации

- `transition` вместо резких изменений
- `cubic-bezier(0.4, 0, 0.2, 1)` для плавности
- Длительность: 0.2–0.5 секунд
- Нет анимаций длиннее 1 секунды для интерактивных элементов

### Темы

- Светлая/тёмная через классы `body.light` / `body.dark`
- Переменные не используются — явные значения для каждой темы
- `transition` на `background-color`, `color`, `border-color`

### Запрещено

- `!important` без крайней необходимости
- Инлайн-стили
- Селекторы глубже 3 уровней
- Универсальный селектор `*` для производительности (кроме сброса)
