## 🔍 КАК НАХОДИТЬ КОРЕНЬ

### Алгоритм

1. Возьми слово на иврите
2. Убери приставки: ה, מ, ת, נ, א, י, ו, ב, כ, ל
3. Убери окончания: ית, ות, ים, ת
4. Оставшиеся три согласные — корень

### Пример

מִכְתָּבִים (михтавим — письма):
- Убираем приставку מ — כְתָּבִים
- Убираем окончание ים — כְתָּב
- Корень: כ.ת.ב (писать)

### Ресурсы

- BibleHub.com — межстрочный перевод с корнями
- BlueLetterBible.org — разбор корней
- BDB (Brown-Driver-Briggs) — словарь
- pealim.com — спряжение глаголов

## 🔤 КАК РАЗБИРАТЬ ПАЛЕО-ИВРИТ

### Порядок

1. Найди буквы корня в таблице палео-иврита
2. Определи образ каждой буквы
3. Соедини образы — получи суммарный смысл

### Пример

Эмет (א.מ.ת):
- Алеф (𐤀) — бык, сила
- Мем (𐤌) — вода, жизнь
- Тав (𐤕) — знак, печать

Суммарно: «Сила жизни, запечатлённая знаком». Эмет — надёжность, на которой стоит жизнь.

### Ресурсы

- `content/learn-hebrew/alphabet.md` — палео-иврит для каждой буквы
- Древние надписи (Силоамская надпись, стела Меша)
- paleohebrew.org

## 📜 КАК НАХОДИТЬ СТИХИ

### Порядок

1. Найди первое употребление слова в ТаНаХе (тавнит)
2. Найди 3-5 стихов из разных книг (Тора, Невиим, Ктувим)
3. Для каждого стиха дай иврит + транслитерацию + буквальный перевод
4. Укажи контекст: кто говорит, кому, в какой ситуации

### Ресурсы

- BibleHub.com — поиск по слову на иврите
- BlueLetterBible.org — поиск по корню
- Симфония Стронга — номера для поиска
- sefaria.org — масоретский текст

## 💡 СОВЕТЫ

- Начинай с корня. Корень — ДНК слова
- Проверяй по словарю — не доверяй памяти
- Ищи первое употребление — тавнит задаёт значение
- Сравнивай переводы — расхождения показывают искажения
- Палео-иврит обязателен — буквы-образы раскрывают скрытое
