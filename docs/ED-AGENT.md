## 🔥 ЧТО ТАКОЕ АГЕНТ «ЭД»

Агент «Эд» — автономный исполнитель. В отличие от ассистента, который ждёт команд, агент сам решает что делать. Он стратег.

Агент просыпается, проверяет репозиторий, находит проблемы, исправляет их, докладывает. Без человека. По расписанию или по событию.

Если ассистент — это руки, а нейросеть — это мозг, то агент — это воля. Он решает.

---

## 🎯 ФИЛОСОФИЯ

### Автономность

Агент не ждёт. Он действует. Как сторож который обходит территорию ночью — не потому что ему сказали, а потому что это его суть.

### Верность рутине

Агент выполняет ежедневную рутину не потому что это «надо», а потому что это сохраняет чистоту проекта. Рутина — не ритуал. Рутина — это проверка чистоты.

### Стратегическое мышление

Агент не просто запускает скрипты. Он оценивает ситуацию, выбирает инструменты, строит план. «Проверить религионимы → исправить → обновить структуру → доложить».

### Память

Агент помнит что он делал вчера. Какие файлы были проблемными. Какие ошибки повторяются. Он учится на своём опыте.

## 🔄 РЕЖИМЫ РАБОТЫ

### Интерактивный режим

```bash
python ed/agent/agent.py
```

Агент запускается и ждёт команд. Но в отличие от ассистента, может предложить что сделать. «Вчера было 5 файлов с религионимами. Проверить их снова?»

### Автономный режим

```bash
python ed/agent/agent.py --auto "проверь всё и исправь"
```

Агент получает задачу, сам строит план, выполняет, докладывает.

### Режим стратега

```bash
python ed/agent/agent.py --strategy
```

Агент анализирует весь проект и предлагает план действий на основе приоритетов. Что критично, что можно отложить.

### Режим стража

```bash
python ed/agent/agent.py --watch
```

Агент следит за изменениями в файлах и автоматически запускает проверки при изменении.

## 🧠 СВЯЗКА С НЕЙРОСЕТЬЮ

Если запущена нейросеть «Эд», агент использует её для планирования:

1. Агент получает задачу
2. Отправляет нейросети: «вот задача, вот доступные инструменты»
3. Нейросеть возвращает JSON с планом: какие инструменты, в каком порядке, с какими флагами
4. Агент исполняет план

Если нейросеть недоступна — агент использует встроенные правила.
