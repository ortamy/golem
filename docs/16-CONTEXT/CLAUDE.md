## СУТЬ
Реконструкция палео-образного мышления, которым обладал древний читатель ТаНаХа.

## ПРАВИЛА РАБОТЫ

### Источник
  1. ТаНаХ на палео-иврите — эталон. Переводы — слои интерпретации.
  2. Палео-иврит первичен. Буква = образ, не звук.
  3. Оригинал — слитный поток согласных. Огласовки, пробелы, кантилляция — интерпретация масоретов (VI-X вв.).
  4. Прасемитский (~3500-3000 до н.э.) — ключ к смыслу до подмен. Если корень есть в аккадском — его значение первично.

### Разбор
  5. Слово: палео-образы → трёхбуквенный корень → прасемитские параллели → смысл.
  6. Стих: иврит + транслитерация + буквальный перевод.
  7. Нет корня в иврите → слово пришлое.

### Фильтр
  8. Греческие философемы отсекать (душа→жизнь, дух→дыхание, вера→верность).
  9. Масоретские правки выявлять (ТМ vs LXX vs Qumran).
  10. Смихут не разрывать.

### Критерий
  11. Палео-образы конкретны. Если сложно — ложь.
  12. Плоды = критерий: жизнь/свобода = от Яхве; смерть/рабство = от системы.
  13. Проверка через Шма: ведёт ли к единству Яхве?

### Действие
  14. ТаНаХ — руководство, не история. Читать вопросом «что делать?».
  15. Глагол первичен. Возвращать глагольность.
  16. Имя = присутствие. Яхве, не Господь. Йешуа, не Иисус.
