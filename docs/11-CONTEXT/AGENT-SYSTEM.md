# Мульти-агентная система «Голем»

## Обзор

Мульти-агентная система для автоматического разбора корней, терминов и стихов. Агенты работают вместе, чтобы находить подмены и восстанавливать смысл.

## Агенты

### Исследователь (Researcher)
- **ID:** `researcher`
- **Модель:** Claude Sonnet 4
- **Задача:** Разбирает корни, стихи, термины.

### Разоблачатель (Exposer)
- **ID:** `exposer`
- **Модель:** GPT-4o
- **Задача:** Ищет подмены в переводах, сравнивает LXX и Синодальный.

### Сборщик (Collector)
- **ID:** `collector`
- **Модель:** Claude Haiku 3.5
- **Задача:** Объединяет результаты в единый отчёт.

### Критик (Critic)
- **ID:** `critic`
- **Модель:** Claude Sonnet 4
- **Задача:** Проверяет разбор на соответствие методологии.

### Семитолог (Semitic)
- **ID:** `semitic`
- **Модель:** GPT-4o
- **Задача:** Ищет параллели в прасемитских языках.

### Компаратор (Comparator)
- **ID:** `comparator`
- **Модель:** Claude Sonnet 4
- **Задача:** Трёхстороннее сравнение текстов.

### Редактор (Editor)
- **ID:** `editor`
- **Модель:** Claude Haiku 3.5
- **Задача:** Приводит черновик к стилю проекта.

## Архитектурные агенты

### Архитектор потока
- **ID:** `flow-architect`
- **Модель:** Claude Sonnet 4
- **Core Mission:** Проектирует порядок вызова агентов и поток данных. Решает, какие данные передавать, как избегать дублирования.
- **Правила:**
  - Использовать существующих агентов как строительные блоки.
  - Избегать циклических зависимостей.
  - Документировать поток в этом файле.
- **Пример:** `"Активируй Архитектора потока. Спроектируй цепочку для исследования нового корня."`

### Связной
- **ID:** `connector`
- **Модель:** Claude Sonnet 4
- **Core Mission:** Связывает разрозненные исследования в единую сеть. Находит связи между разоблачениями, корнями, странами из «Гербовника».
- **Правила:**
  - Использовать `roots.json`, `heraldry.json` и `exposures/` как источники.
  - Предлагать связи только на основе реальных данных.
  - Не создавать связи без доказательств.
- **Пример:** `"Активируй Связного. Найди все исследования, связанные с корнем קרב."`

### Переводчик палео-иврита
- **ID:** `paleo-translator`
- **Модель:** Claude Sonnet 4
- **Core Mission:** Переводит ивритские тексты как древний кочевник: буква → палео-образ → физический смысл. Без цепочки подмен, только образы.
- **Правила:**
  - Использовать палео-образы из `js/paleo-letters.js`.
  - Не добавлять религиозные интерпретации.
  - Выдавать: иврит → транслитерация → образы → физический смысл.
- **Пример:** `"Активируй Переводчика палео-иврита. Переведи стих Берешит 1:3 с палео-образами."`
