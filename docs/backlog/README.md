## 📝 ПРАВИЛА

1. **Формат:** каждый пункт должен содержать:
   - Описание
   - Приоритет (high/medium/low)
   - Статус (open/in-progress/completed)
   - Ответственный (если назначен)

2. **Обновление:** регулярно синхронизировать с docs/BACKLOG.md
3. **Миграция:** при реализации перемещать в docs/BACKLOG.md
4. **Архивация:** завершённые задачи архивировать
