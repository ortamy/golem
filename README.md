## 🎯 ДЛЯ КОГО ЭТОТ РЕПОЗИТОРИЙ

- **Для ищущих истину** — тех, кто чувствует, что традиционные переводы и доктрины лгут, и хочет вернуться к корням.
- **Для исследователей** — кто готов проверять, задавать вопросы, не принимать на веру.
- **Для ремесленников духа** — кто хочет не просто «знать», а **делать**: очищать язык, восстанавливать смыслы, строить автономию.
- **Для Свидетелей** — тех, кто готов свидетельствовать об истине, даже если это идёт против течения.

**Этот репозиторий не для всех.** Он для тех, кто устал от лжи и готов идти путём Яхве.
