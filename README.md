# 📜 ГОЛЕМ — СВИДЕТЕЛЬ ИСТИНЫ

> ⚠️ **Единственный источник истины — масоретский текст ТаНаХа на иврите.**

---

## 🔥 ЧТО ЭТО

«Голем» — восстановление того, что было украдено Римом, затемнено раввинами и искажено переводчиками.

Мы не создаём новую религию. Мы возвращаемся к тому, что было от начала.

---

## 🧭 БЫСТРЫЙ СТАРТ

- **Веб-интерфейс:** [ortamy.github.io/golem](https://ortamy.github.io/golem)
- **Локально:** `cd web && node server.js` → [localhost:8080](http://localhost:8080)
- **Главное меню:** `python tools/golem.py`
- **Пульт управления:** `CONTROL.md`
