## 🔥 ЧТО ТАКОЕ CLINE

ИИ-ассистент в VS Code. Читает файлы, пишет код, запускает команды. Работает через DeepSeek API.

Понимает русский. Не помнит историю между чатами.

---

## 🚀 УСТАНОВКА

1. VS Code → Extensions → **Cline** → Установить
2. Иконка Cline в боковой панели
3. Settings → API Provider → **DeepSeek**
4. Вставить API ключ
5. Model → **deepseek-chat**

---

## 📋 КАК ДАВАТЬ ЗАДАЧИ

### Хорошая задача (одна, конкретная, один файл)

«Обнови строку 7 в файле tools/utils/rename-script.py — замени Path(__file__).parent на Path(__file__).parent.parent»

### Плохая задача (размытая, много файлов)

«Обнови все файлы в проекте»

### Правила:
- Одна задача = одно сообщение
- Указывай конкретный файл
- Пиши что заменить и на что
- Не проси «посмотри и предложи» без конкретики

---

## 🎯 ЧТО CLINE ДЕЛАЕТ ЛУЧШЕ ВСЕГО

- Правит код в одном файле
- Добавляет новые функции
- Создаёт скрипты по описанию
- Находит и заменяет текст
- Читает файлы и отвечает на вопросы

---

## ⚠️ ЧТО CLINE ДЕЛАЕТ ПЛОХО

- Массовые операции (>10 файлов) — медленно
- Помнит контекст только в рамках одного чата
- Не знает историю проекта
- Иногда удаляет нужное (проверяй изменения)

---

## ⌨️ ГОРЯЧИЕ КЛАВИШИ

- `Ctrl+L` — открыть чат Cline
- `Ctrl+Shift+I` — действие с выделенным кодом

---

## 🔄 КОГДА НАЧИНАТЬ НОВЫЙ ЧАТ

- Новая задача не связана с предыдущей
- Модель путается или повторяется
- Контекст перегружен (>50 сообщений)
- Меняешь модель (flash → chat)

---

## 💡 СОВЕТЫ

- Перед правкой попроси показать что будет изменено
- Проверяй diff перед сохранением
- Делай коммит в Git перед большими правками
- Если Cline удалил не то — откати через Git
- Не давай задачи требующие знания истории проекта
