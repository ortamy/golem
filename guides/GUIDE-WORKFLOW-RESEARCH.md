## 🔥 8 ШАГОВ ОТ ИДЕИ ДО ПУБЛИКАЦИИ

### Шаг 1: Идея

Откуда берём тему:
- Из личного изучения ТаНаХа
- Из разговора с Эдом (нейросетью)
- Из списка идей в `docs/IDEAS.md`
- Из незаполненных файлов (`check-empty-files.py`)

Результат: одно предложение — о чём будет исследование.

---

### Шаг 2: Проверка на дубликат

```bash
python tools/utils/search.py "ключевое слово"
```

Убедиться что такого файла ещё нет в `researches/` и `terminology/`.

---

### Шаг 3: Создание черновика

Создать файл в `drafts/` с временным названием.

Использовать шаблон: `instructions/templates/research-template.md`

Обязательно заполнить:
- Заголовок H1 (эмоджи + КАПС)
- Блок метаданных
- Раздел «Введение»

---

### Шаг 4: Наполнение

Работа с текстом:
- Иврит + транслитерация + перевод для каждого стиха
- Палео-иврит (разбор букв)
- Корень слова
- Контекст ТаНаХа (минимум 3-5 стихов)
- Связь с Машиахом (если применимо)
- Практическое применение сегодня

---

### Шаг 5: Аудит (4 этапа)

```bash
# 1. Проверка на религионизмы
python tools/checkers/check-religionisms.py --fix

# 2. Проверка exposure-критериев
python tools/checkers/check-exposure.py

# 3. Проверка фактов
# Вручную по чек-листу: instructions/checkers/check-factcheck.md

# 4. Проверка согласованности
python tools/checkers/check-consistency.py
```

---

### Шаг 6: Выбор папки и имени

Куда положить:
- Разбор слова → `terminology/`
- Разбор явления/системы → `researches/`
- Практическое → `researches/practices/`
- Языковое → `researches/language/`
- Танахическое → `researches/tanakh/`

Имя файла: иврит латиницей через дефис. Пример: `mashal-melachim.md`

---

### Шаг 7: Перекрёстные ссылки

Добавить ссылки на связанные файлы в метаданные.

```bash
python tools/generators/generate-related-links.py
```

---

### Шаг 8: Публикация

```bash
# Переместить из drafts в нужную папку
mv drafts/новый-файл.md researches/tanakh/имя-файла.md

# Обновить структуру
python tools/sync/sync-structure.py

# Обновить files.json для веб-интерфейса
python tools/generators/generate-files-json.py

# Проверить ссылки
python tools/checkers/check-links.py

# Коммит
git add . && git commit -m "Новое исследование: название"
git push
```
