## 🔥 ВИДЕНИЕ

Давар (דָּבָר) — это язык, в котором слово и дело — одно.

Обычный язык программирования описывает реальность. Давар создаёт её. Обычный код говорит компьютеру: «сделай это». Давар говорит реальности: «будь».

Это не магия. Это архитектура мышления. Язык, в котором синтаксис отражает ивритскую картину мира: конкретную, образную, без абстракций. Где переменные — не `x`, а живые существа. Где функции — не `void`, а творящие глаголы. Где ошибки — не `exception`, а промах (хэт), который можно исправить.

Изучая Давар, человек учится мыслить как ТаНаХ. Писать на Даваре — значит исследовать Писание. Компилировать Давар — значит видеть, как слово становится делом.

## 📜 ОСНОВНЫЕ ПРИНЦИПЫ

### Слово и дело — одно
В Даваре нет разницы между объявлением и действием. `light.be()` — это и команда, и свершившийся факт. Никакого зазора. Как в Берешит: «И сказал Элохим: да будет свет. И стал свет».

### Конкретика вместо абстракции
Нет типов `int`, `string`, `boolean`. Есть `свет`, `вода`, `земля`, `дыхание`, `имя`, `день`, `путь`. Нет переменных `x`, `y`, `tmp`. Есть `адам`, `хава`, `дерево_жизни`, `ковчег`.

### Промах вместо ошибки
Давар не крашится. В нём нет `exception`. Есть `хэт` — промах. Программа не падает — она возвращает возможность тшувы. `sin.commit()` → `Error: sin is not recognized. Did you mean 'return'?`

### Суббота — остановка
Давар знает ритм творения. У каждого цикла есть предел. День первый, день второй... день седьмой — `shabbat()`. После шести итераций цикл останавливается. Не по ошибке. По замыслу.

### Отделённость
Давар различает. Нельзя смешать свет и тьму. Нельзя объявить `light.dark()` — это не скомпилируется. Таhор и тамэ — разные типы. Пригодное и разрушительное — разные ветки кода.

## 🛠️ СИНТАКСИС

### Переменные

Переменная — это не ячейка памяти. Это имя. А имя — это суть.

```

адам: земля + дыхание
хава: адам.ребро.взятое()
дерево_жизни: дерево.в_центре(сад)

```

Каждая переменная знает своё происхождение. Адам — из земли и дыхания. Хава — из ребра Адама. Нельзя создать переменную из ничего.

### Функции

Функция — это не `def` и не `function`. Это глагол творения.

```

да_будет(свет):
тьма.расступиться()
пространство.наполнить(свет)
return "тов"

свет.быть()  # вызывает да_будет(свет)

```

Функция не возвращает `null`. Она возвращает либо то, что создала, либо «тов». Либо промах.

### Условия

Условия — это выбор. Не `if/else`, а «если послушаешь — благословение, если нет — проклятие».

```

если народ.слушает(тора):
земля.давать_плод()
небо.давать_дождь()
если_нет:
земля.иссушить()
небо.закрыть()

```

Выбор всегда есть. И он всегда имеет последствия. Не абстрактные. Конкретные.

### Циклы

Циклы — это ритм творения. День первый, день второй... день седьмой.

```

повторить 6 раз (день):
творить()
разделять()
называть()
return "тов"

шаббат():
остановиться()
дышать()
помнить()

```

Цикл не бесконечен. У него есть предел. Суббота — это выход из цикла. Не ошибка. Завершение.

### Работа с данными

Данные — это не абстрактные структуры. Это живые существа, деревья, воды.

```

сад.посадить(дерево_жизни)
сад.посадить(дерево_познания)
адам.поместить_в(сад)
адам.назвать(животные)

```

Нет `array.push()`. Нет `object.assign()`. Есть `посадить`, `поместить`, `назвать`. Конкретные действия с конкретными вещами.

## 🏺 ТИПЫ ДАННЫХ

### Базовые типы

- **свет** — то, что отделено от тьмы. Видимое. Истинное.
- **тьма** — то, что было до света. Потенциал. Не зло.
- **вода** — текучее. Живое. Может быть разделено.
- **земля** — твёрдое. Основание. То, из чего создан Адам.
- **дыхание** — то, что оживляет. Невидимое, но реальное.
- **имя** — суть вещи. Не ярлык. То, что определяет.
- **день** — единица времени. Состоит из вечера и утра.
- **путь** — направление. Выбор. То, по чему идут.
- **слово** — действие. Сказанное и сделанное одновременно.

### Составные типы

- **человек** — земля + дыхание. Живое существо.
- **дерево** — плод + семя. То, что приносит жизнь.
- **союз** — обязательство + верность. Брит.
- **Дом** — место + присутствие. Микдаш.

## ⚖️ ПРАВИЛА

- **Нельзя лгать.** `утверждение.ложь()` не компилируется. Язык не поддерживает ложь как операцию.
- **Нельзя смешивать несмешиваемое.** `свет + тьма` — ошибка типа. `таhор + тамэ` — ошибка отделённости.
- **Нельзя создавать из ничего.** Всё имеет источник. `переменная.из_ничего()` — ошибка.
- **Нельзя отменить тшуву.** Даже после промаха можно вернуться. `возврат()` всегда доступен.
- **Нельзя работать в субботу.** `шаббат.работать()` — ошибка ритма.
