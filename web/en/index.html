What is Project "Golem"?

The TaNaKh is not the Bible. The Bible is a translation — rewritten, edited, and adapted to serve empires. Project "Golem" returns to the Hebrew original to see the text as it was meant to be.

The system has replaced the Name Yahveh with "Lord", erased the distinction between "el" (strength, power) and "elohim" (judges, rulers), turned "mashiach" (anointed one) into a title, and "torah" into "law". We restore every word.

This is not a religion. It is a return to the Source. To the Truth that speaks for itself — in the language in which it was given.

Key Truths

𐤉
Yahveh is a Name, not a title. "Lord", "God", "Almighty" are replacements. The Name Yahveh (יהוה) is personal, given by Himself. To speak or replace it is a choice. Truth demands precision.
𐤀
Yahveh is one. Shema, Israel: Yahveh is our Elohim, Yahveh is one. Not a trinity, not a pantheon, not forces of nature. One Source of all that exists.
𐤌
Mashiach is an anointed one, not a title. Mashiach (משיח) is one who is anointed for service: king, kohen, prophet. It is a function, not a name. Understanding this changes everything.
𐤕
Torah is instruction, not "law". Torah (תורה) comes from the root ירה — "to direct, to teach". It is not a code of rules but a guidepost. Torah is light, not a yoke.
𐤔
Teshuvah is return, not "repentance". Teshuvah (תשובה) is not guilt — it is action: returning to where you belong. To Yahveh, to Truth, to yourself.
