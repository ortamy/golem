## 📝 ПРАВИЛА

1. **Архивация:** сюда перемещаются неиспользуемые файлы
2. **Организация:** файлы должны быть отсортированы по категориям

4. **Удаление:** периодически проверять и удалять не нужное

## 🛠 ИСПОЛЬЗОВАНИЕ

### Добавление в архив
```bash
# Переместить файл в архив
mv old-file.png archive/
```

### Просмотр архива
```bash
# Просмотреть содержимое
ls -la archive/
```

### Восстановление из архива
```bash
# Вернуть файл из архива
mv archive/old-file.png .
```
