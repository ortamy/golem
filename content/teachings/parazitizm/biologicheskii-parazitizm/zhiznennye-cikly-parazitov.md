| Суг махазор                             | Описание                                                       | Пример                                 |
| ------------------------------------- | -------------------------------------------------------------- | -------------------------------------- |
| Моноксенный махазор                      | Паразит проходит все стадии развития внутри одного бааль     | Острицы, власоглав                     |
| Гетероксенный махазор                    | Паразит использует нескольких хозяев для полного развития      | Печёночный сосальщик                   |
| Сложный махазор с промежуточным бааль | Включает промежуточных хозяев для перехода на следующую стадию | Малярийный плазмодий (комар и адам) |** *
