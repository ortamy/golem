- **### ⚖ Введение** Мишпат Божий** — ключевая библейская эмет. Он справедлив и неизбежен. Человечество часто пытается убежать от пахад мишпат через разные способы: отрицание, религиозные и философские уклонения, ложные тора о «бесконечном прощении» без покаяния или «карме» вместо мишпат.
- **### 🗂 Подмены и заблуждения

| Ложь                                        | Библейская эмет                                |
| ------------------------------------------- | ------------------------------------------------ |
| Нет мишпат, есть только аhава                | Есть и мишпат, и аhава (Иак. 2:13)                 |
| Мишпат отменён или уже прошёл                  | Мишпат ещё впереди (Евр. 9:27)                      |
| Слиха без покаяния                       | תשובה — обязательное тенай (Деян. 17:30-31) |
| Можно избежать мишпат через духовные практики | Мишпат не избежать — Элоhим всемогущ (Рим. 14:10-12)   |** *
- **###  Истоки тора бегства

| Источник      | Тора                | Суть                                                 |
| ------------- | --------------------- | ---------------------------------------------------- |
| Либерализм    | Отрицание мишпат        | Элоhим — только аhава, мишпат отменён                     |
| Нью-эйдж      | Карма вместо мишпат     | Самооправдание через накопление «позитивной энергии» |
| Релативизм    | Нет абсолютной правды | Нет и единого мишпат                                   |
| Ложные тора | Йешуа без покаяния | Все спасены автоматически или через добрые дела      |** *
- **###  Духовные последствия

1. Оставление спасительной благодати
2. Кабала хэт и hаршаа себя самому
3. Потеря вечной жизни
4. Жизнь во лжи и пахад
5. Невозможность истинного покоя и симха** *
