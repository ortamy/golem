# 🏺 Прообразы вещей

📌 Эти образы — не просто инструменты или метафоры.\
Они указывают на:

* характер человека,
* действие Бога,
* труд за плату → עֲבוֹדָה — служение (авода, עֲבוֹדָה) Духа,
* юридический процесс → מִשְׁפָּט — правосудие. Подмена: восстановление справедливости заменено на состязательный процесс (мишпат, מִשְׁפָּט) и милость Божью.
