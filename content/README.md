## 🔗 СВЯЗИ

### Внутренние связи
- `terminology/` ↔ `teachings/` - термины используются в учениях
- `tanakh/` ↔ `bashah/` - Танах и Брита Хадаша
- `researches/` ↔ `tanakh/` - исследования ссылаются на Танах
- `tzel/` ↔ `tanakh/` - типы из Танаха

### Внешние связи
- `docs/GLOSSARY.md` - глоссарий ссылается на terminology/
- `instructions/dictionaries/` - словари для проверок
- `tools/checkers/` - скрипты проверки контента

## 📝 ПРАВИЛА

1. **Метаданные:** каждый файл должен содержать блок метаданных
2. **Именование:** lowercase, дефисы вместо пробелов
3. **Ссылки:** использовать относительные пути
4. **Проверка:** перед коммитом запускать чекеры
